**Spec:** Write a Python function merge_sorted(a, b) that merges two sorted lists into one sorted list without using sorted() or list.sort(). Both input lists contain integers and are already sorted ascending. The function must run in O(n+m) time.

## Generated Code

```python
def merge_sorted(a, b):
    result = []
    i = j = 0
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            result.append(a[i])
            i += 1
        else:
            result.append(b[j])
            j += 1
    result.extend(a[i:])
    result.extend(b[j:])
    return result
```

## Explanation

The function uses the classic two-pointer merge: it walks both lists simultaneously with index variables `i` and `j`, always appending whichever of the two current elements is smaller, then advancing that list's pointer. This guarantees each element is visited exactly once, giving O(n+m) time. The two `extend` calls at the end handle the edge case where one list is exhausted before the other — whatever remains in either list is already sorted, so it can be appended wholesale without further comparison. This also correctly handles inputs where one or both lists are empty, since the `while` loop simply never executes and the extends copy the non-empty list (or nothing) directly.
