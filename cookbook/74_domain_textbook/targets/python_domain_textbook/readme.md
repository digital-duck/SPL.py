# build_micro_textbook (Python/domain_textbook notebook)

Generated by `splc --lang python/domain_textbook`

Unlike `python/linalg`/`python/intro_geometry`, this notebook does not
hardcode a domain — it resolves `@domain_yaml` (e.g. `linalg_graph.yaml`
or `geometry_graph.yaml`) through `graph_lib.load_domain()` at *run time*,
so the SAME compiled notebook can teach either domain depending on input.

## Install

```bash
pip install networkx jupyter sympy pyyaml
# graph_lib.py + {domain}_graph.yaml must be importable;
# set DOMAIN_TEXTBOOK_GRAPH_DIR if needed
```

## Run

```bash
DOMAIN_YAML=linalg_graph.yaml jupyter notebook build_micro_textbook_python_domain_textbook.ipynb
# or: DOMAIN_YAML=geometry_graph.yaml jupyter nbconvert --to notebook --execute build_micro_textbook_python_domain_textbook.ipynb
```
