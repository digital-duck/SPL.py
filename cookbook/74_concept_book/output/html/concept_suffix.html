Suffix

A suffix is a sequence of characters that follows another sequence of characters in a word or phrase. In linguistic terms, it refers to the part of a word that comes after the root and prefix (if present). For example, in the word "unpredictable," "un" is the prefix and "able" is the root with a suffix.

Worked Example

Consider the following words:

  • cat
  • happy
  • running

    • We can identify the roots by looking at the base part of each word. The roots are "cat," "hap," and "run." To form these words, we have added prefixes ("un-" for "unhappy") and suffixes.

    | Word | Root | Prefix/Suffix | | --- | --- | --- | | cat | cat | - | | happy | hap | un- | | running| run | ing- |

    Key Theorem

    The suffix can be classified into different types, such as:

  • -able (e.g. happy)
  • -ing (e.g. running)
  • -less (e.g. unhappy)

    • These classifications are based on the sound and meaning of the suffix.

    Lab Cell (SymPy)

    import sympy as sp

# Define the variables
x = sp.symbols('x')

# Define a function to calculate the root with prefix/suffix
def calculate_word(root, prefix=None, suffix=None):
    if prefix:
        return f"{prefix}{root}"
    elif suffix:
        return f"{root}{suffix}"
    else:
        return root

# Calculate and print the words using the function
print(calculate_word('cat'))  # cat
print(calculate_word('happy', 'un-'))  # unhappy
print(calculate_word('run', '', '-ing'))  # running
