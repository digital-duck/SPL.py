Transcendental functions are a class of mathematical functions that cannot be expressed as finite combinations of addition, subtraction, multiplication, division, exponentiation, or roots. These functions often involve trigonometric expressions and have numerous applications in physics, engineering, and other fields.

Definition

A transcendental function is defined as a function that can be written in the form:

$$f(z) = \sum_{i=1}^{n} c_i e^{k_i z},$$

where $c_i$ are non-zero constants, $z$ is a complex variable, and $e^z$ is the exponential function. This definition encompasses many common transcendental functions, including trigonometric functions, inverse hyperbolic functions, and logarithmic functions.

Worked Example

Consider the following example of a transcendental function:

$$f(z) = e^{z} + \sin z.$$

To evaluate this function at $z = 1$, we can use the definitions of the exponential and sine functions:

import sympy as sp

z = sp.symbols('z')

# Define the transcendental function
f = sp.exp(z) + sp.sin(z)

# Evaluate the function at z = 1
result = f.subs(z, 1)
print(result)
