Prefix Root Composition

Prefix root composition is a fundamental operation in algebraic manipulations that allows for the simplification of expressions by combining two operations: prefixing and rooting.

Definition

Given an expression \(f(x)\) and a polynomial \(g(x)\), the prefix root composition of \(f(x)\) with respect to \(g(x)\) is defined as: \[ \text{pr}(f(x), g(x)) = f(g(x)) \]

In other words, we first apply the root operation to the input of function \(f\), and then apply function \(f\) to the result. This operation combines the concepts of function composition and radical expressions.

Worked Example

Consider the expression \(f(x) = x^2 + 3x - 4\) and the polynomial \(g(x) = x - 1\). We can perform prefix root composition as follows:

\[ \text{pr}(f(x), g(x)) = f(g(x)) = (g(x))^2 + 3(g(x)) - 4 \]

Substituting \(g(x)\), we get: \[ \text{pr}(f(x), g(x)) = ((x-1)^2) + 3((x-1)) - 4 \] \[ \text{pr}(f(x), g(x)) = (x^2 - 2x + 1) + 3x - 3 - 4 \] \[ \text{pr}(f(x), g(x)) = x^2 + x - 6 \]

Key Theorem

The prefix root composition is a one-to-one correspondence between the set of all expressions of \(f(g(x))\) and the set of all expressions of the form \(a(b(c))^n\), where \(a, b, c, n \in \mathbb{R}\).

Lab Cell (SymPy)

from sympy import symbols, expand

# Define variables
x = symbols('x')

# Define functions
f = x**2 + 3*x - 4
g = x - 1

# Perform prefix root composition
result = f.subs(x, g)

# Expand the result
expanded_result = expand(result)
print(expanded_result)
