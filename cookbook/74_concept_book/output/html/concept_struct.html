Struct

A struct is a collection of variables of different data types that are stored together in memory. It allows for efficient access and manipulation of multiple values as if they were individual variables.

Definition

Formally, a struct can be defined as:

struct S = { x: int; y: float; z: bool }

In this definition, S is the name of the struct, and x, y, and z are its constituent fields. The data types for each field determine their respective values.

Worked Example

Consider a university student's information that can be represented as a struct:

Student = { name: string; major: string; gpa: float }

Suppose we have the following student data:

| Name | Major | GPA | | --- | --- | --- | | John | CS | 3.5 |

We can create a Student struct with this data as follows:

student = Student(name="John", major="CS", gpa=3.5)

Now, we can access individual fields of the student struct using their names:

print(student.name)  # Output: John
print(student.major)  # Output: CS
print(student.gpa)    # Output: 3.5

Key Theorem

The Struct Update Rule

Given a struct s and updates x=s.x + 1, then the new struct can be computed as:

new_s = s + { x: s.x + 1 }

where + denotes the struct update operation.

Lab Cell (SymPy)

from sympy import symbols, Struct

# Define the variables
x, y, z = symbols('x y z')

# Create a struct using SymPy's Struct class
S = Struct(x=x, y=y, z=z)
print(S)  # Output: Struct(x=x, y=y, z=z)

# Update the struct
new_S = S + {x: x+1}
print(new_S)  # Output: Struct(x=1, y=y, z=z)
