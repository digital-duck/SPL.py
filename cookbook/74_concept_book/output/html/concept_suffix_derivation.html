Suffix Derivation

Suffix derivation is a fundamental concept in mathematical analysis that enables us to construct new functions from existing ones by applying specific rules. In this context, we will focus on the suffix derivation technique used to generate derivatives of functions.

Definition

Given a function $f(x)$ and a polynomial $p(x) = \sum_{i=0}^{n} c_i x^i$, the suffix derivative of $f(x)$ with respect to $p(x)$ is denoted as $\frac{d}{dx}[f(x)p(x)]$. It can be computed using the product rule for differentiation.

Worked Example

Consider the function $f(x) = e^x$ and the polynomial $p(x) = 3x + 2$. We want to find the suffix derivative of $f(x)$ with respect to $p(x)$.

We can write the expression $\frac{d}{dx}[f(x)p(x)]$ using the product rule as: \begin{align*} \frac{d}{dx}[e^x(3x+2)] &= \left(\frac{d}{dx}f(x)\right)(3x+2) + f(x) \left(\frac{d}{dx}(3x+2)\right) \\ &= e^x (3x+2) + e^x (3) \\ &= e^x(3x+5) \end{align*}

Key Theorem

The suffix derivative of a function $f(x)$ with respect to a polynomial $p(x) = \sum_{i=0}^{n} c_i x^i$ is given by the formula:

$$ \frac{d}{dx}[f(x)p(x)] = \left(\frac{df(x)}{dx}\right)(p(x)) + f(x) \left(\frac{dp(x)}{dx}\right) $$

Lab Cell (SymPy)

import sympy as sp

# Define the variables and functions
x, y = sp.symbols('x y')
f = sp.exp(x)
p = 3*x + 2

# Compute the suffix derivative of f(x) with respect to p(x)
suffix_derivative = sp.diff(f*p, x)

print(suffix_derivative)

This code computes the suffix derivative of $f(x)$ with respect to $p(x)$ and prints the result.
