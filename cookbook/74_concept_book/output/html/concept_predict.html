Predict

Predict in the context of mathematical modeling refers to the process of estimating or forecasting future events or behavior based on past data, trends, and patterns. In various fields such as economics, finance, epidemiology, and climate science, prediction is a crucial tool for making informed decisions, predicting outcomes, and understanding complex systems.

Definition

Prediction can be viewed as an optimization problem where the goal is to minimize the difference between observed and predicted values. Mathematically, this can be formulated as:

$$\min_{\theta} \sum_{i=1}^{n} (y_i - f(x_i, \theta))^2$$

where $y_i$ represents the observed value at time step $i$, $x_i$ is the input or feature vector, $\theta$ is the model parameter, and $f(x_i, \theta)$ is the predicted output.

Worked Example: Simple Linear Regression

Consider a scenario where we want to predict house prices based on the number of bedrooms. We collect data on 10 houses with their corresponding prices and bedroom counts:

| Bedrooms | Price | | --- | --- | | 2 | $150,000 | | 3 | $200,000 | | 4 | $250,000 | | ... | ... |

We fit a simple linear regression model to the data:

$$y = \beta_0 + \beta_1 x$$

where $x$ is the number of bedrooms and $\beta_0$ and $\beta_1$ are model parameters.

Using ordinary least squares (OLS), we find that $\hat{\beta}_1 = 50,000$ and $\hat{\beta}_0 = -25,000$. We then use this model to predict the price for a house with 3 bedrooms:

$$\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x = -25,000 + 50,000(3) = $75,000$$

Key Theorem: Bayes' Rule

Bayes' rule provides a framework for updating the probability of a hypothesis based on new evidence. In the context of prediction, Bayes' rule can be used to update the predictive distribution of a model as more data becomes available.

Let $\pi_0$ be the prior probability of the null hypothesis $H_0$, and let $\mathcal{D}_i$ be the observed data at time step $i$. Then, Bayes' rule states:

$$\pi_{i+1}(|H_0|) = \frac{\pi_i(|H_0|)P(D_i|H_0)}{\sum_{j=0}^J \pi_i(|H_j|)P(D_i|H_j)}$$

where $\pi_i(|H_k|)$ is the posterior probability of hypothesis $H_k$ at time step $i$, and $P(D_i|H_k)$ is the likelihood of observing data $D_i$ under hypothesis $H_k$.

import numpy as np
from sympy import symbols, Eq, solve

# Define variables
x = symbols('x')
y = symbols('y')

# Simple linear regression model
beta0, beta1 = symbols('beta0 beta1')
model = Eq(y, beta0 + beta1 * x)

# Observed data
data_x = np.array([2, 3, 4])
data_y = np.array([150000, 200000, 250000])

# Ordinary least squares (OLS) estimation
X = np.vstack([np.ones(len(data_x)), data_x]).T
theta = solve(Eq(X.T @ X * theta - X.T @ y, 0), symbols('beta0 beta1')).values

print("Estimated model parameters:")
print(f"beta0: {theta[0]}")
print(f"beta1: {theta[1]}")

# Predict house price for a house with 3 bedrooms
x_new = 3
y_pred = theta[0] + theta[1] * x_new
print(f"Predicted house price: ${y_pred:.2f}")
