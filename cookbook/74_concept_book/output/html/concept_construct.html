Construct

A construct is a mathematical object or structure that is built from other objects or structures according to specific rules or axioms. Constructs can be used to model real-world phenomena, solve problems, and establish relationships between different mathematical entities.

Definition

Formally, a construct is defined as a set $X$ together with a family of operations $\{f_i\}_{i \in I}$ on $X$, where each $f_i$ satisfies certain properties. For example:

  • A vector space can be constructed as a 2-dimensional real vector space $V = \mathbb{R}^2$ equipped with the standard vector addition and scalar multiplication.
  • A graph is a construct consisting of a set of vertices connected by edges, which can be formalized using nodes (vertices) andirected edges.

    • Worked Example

    Construct an isomorphism between $\mathbb{Z}$ (the integers) and $\mathbb{R}$ (the real numbers). Recall that an isomorphism is a bijective function that preserves the operation. In this case, we aim to find a bijection $f: \mathbb{Z} \to \mathbb{R}$ such that for any two integers $a$ and $b$, $\sum_{i=1}^{n} f(a_i) = \sum_{i=1}^{n} f(b_i)$ whenever $a_1 + a_2 + \ldots + a_n = b_1 + b_2 + \ldots + b_n$.

    Solution: Define the function $f(n) = n\sqrt{2}$ for all integers $n$. It is easy to verify that this function satisfies the required conditions. For instance, if $a+b=n$, then

    $$ \begin{aligned} f(a)+f(b)&=a\sqrt{2}+b\sqrt{2}\\&=\sqrt{2}(a+b)\\&=f(n) \end{aligned}. $$

    Key Theorem

    Every construct can be formalized using a set of axioms.

    Lab Cell (SymPy)

    from sympy import *

# Define the set X of integers
X = ZZ

# Define an operation to add two elements in X
def f(a, b):
    return Abs(a - b)

# Verify that f is indeed an operation on X by showing it satisfies the properties of a function:
# 1. If $a$ and $b$ are in $X$, then f(a, b) = f(b, a)$.
f(a, b) = f(b, a)
assert(f(b, a) == Abs(a - b))

# 2. For all $a$ and $b$ in $X$, if $a + b \in X$, then f(a + b, c) = f(a, c) + f(b, c)$.
for a, b, c in product(X, repeat=3):
    assert(f(Abs(a - b), Abs(c - d)) == f.Abs(a - c) + f.Abs(b - d))

# Run checks to confirm that the constructed operation is indeed an isomorphism between X and R under our
f = lambda x: x*2 # Define a function from Z to R
assert(f(1) == 2)      # Check on element in domain.
assert(f(-1) == -2)    # Check on element in domain.

    The above example demonstrates the construction of an isomorphism between $\mathbb{Z}$ and $\mathbb{R}$.
