Re-

Reconstruction of functions is a crucial concept in calculus, particularly when dealing with piecewise functions or functions with gaps.

Definition

Given a function $f: \mathbb{R} \rightarrow \mathbb{R}$ and a non-empty set $\Phi$ of points that are not in the domain of $f$, the reconstruction of $f$ is denoted by $\tilde{f}$, and defined as:

$$ \begin{array}{ll} f(x), & \text{if } x \notin \Phi \\ L_x, & \text{if } x \in \Phi \end{array} \right.$$ where $L_x$ is a linear function that interpolates $f$ at the points in $\Phi$. ### Worked Example Consider the function $f(x) = \begin{cases} 0, & \text{if } x < 1 \\ 2x - 1, & \text{if } x \geq 1. \end{cases}$ and let $\Phi = \{1/2\}$. Then, the reconstruction of $f$ is: $$

\begin{array}{ll} 0, & \text{if } x < 1 \\ 2x - 1, & \text{if } x \in [0, 1] \cup (1, 2] \\ L_{1/2}, & \text{if } x = 1/2 \end{array} \right.$$

where $L_{1/2}$ is the linear function that interpolates $f$ at $x=1/2$. Solving for $L_{1/2}$, we find:

$$L_{1/2}(x) = f(0) + \frac{f(1) - f(0)}{1-0} (x-0) = 0 + \frac{1}{2} x = \frac{x}{2}.$$

Key Theorem

The reconstruction theorem states that if $f: [a, b] \rightarrow \mathbb{R}$ is a function with a non-empty set of discontinuities $\Phi$, then the reconstruction $\tilde{f}$ is a piecewise linear function on $[a, b]$.

Lab Cell (SymPy)

from sympy import symbols, Piecewise

x = symbols('x')

# Define the original function f(x)
def f(x):
    if x < 1:
        return 0
    else:
        return 2*x - 1

# Define the linear interpolation L_x at point x=1/2
L_x = (x-1)/3

# Define the reconstruction of f(x) using a piecewise function
tilde_f = Piecewise((0, x < 1), (2*x - 1, x >= 1))

print(tilde_f)

This code defines the original function $f(x)$ and the linear interpolation $L_x$ at point $x=1/2$, then uses SymPy's Piecewise function to define the reconstruction $\tilde{f}$ of $f(x)$.
