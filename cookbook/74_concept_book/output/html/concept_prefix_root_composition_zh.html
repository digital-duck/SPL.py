Prefix Root Composition

定义 设 $f(x)$ 和 $g(x)$ 是两个可交合的实值函数。函数复合 $f \circ g$ 定义为 $(f \circ g)(x) = f(g(x))$。当我们考虑“根式前缀复合”（Prefix Root Composition）时，我们关注的是对整个复合函数进行开方（或取 $n$ 次方根）的操作。形式上，如果定义了一个操作 $\mathcal{R}_n(\cdot)$ 为取 $n$ 次方根，那么一个带有根式的复合函数可以记为：

$$h(x) = \sqrt[n]{(f \circ g)(x)} = \left((f \circ g)(x)\right)^{1/n} = (f(g(x)))^{1/n}$$

其中 $n$ 是一个正整数。理解这个结构的关键在于，根号操作 $\mathcal{R}_n(\cdot)$ 必须应用于复合函数 $f(g(x))$ 的整个输出值。在进行此类分析时，我们通常要求 $f(g(x)) \ge 0$，以确保实数域内定义的有效性。

例题 考虑以下两个函数：$f(u) = u + 2$ 和 $g(x) = x^2 - 1$。我们想计算 $\sqrt{f(g(x))}$。首先进行复合，得到 $(f \circ g)(x) = f(g(x)) = (x^2 - 1) + 2 = x^2 + 1$. 然后取平方根：

$$h(x) = \sqrt{(x^2 + 1)} = (x^2 + 1)^{1/2}$$

我们可以使用链式法则求导。令 $y = x^2+1$，则 $\frac{dy}{dx} = 2x$。由于 $h(x) = y^{1/2}$，根据复合函数求导法则：

$$\frac{dh}{dx} = \frac{d}{dx}(y^{1/2}) = \frac{1}{2\sqrt{y}} \cdot \frac{dy}{dx} = \frac{1}{2\sqrt{x^2+1}} \cdot 2x = \frac{x}{\sqrt{x^2+1}}$$
