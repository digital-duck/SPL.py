Transportable

A transportable is a mathematical object that can be mapped from one structure to another while preserving its essential properties. In other words, a transportable is an equivalence class of structures that are "essentially similar" in some sense.

Definition

Given two algebraic structures $A$ and $B$ of the same type, we say that $A$ is transportable to $B$ if there exists a bijection $\phi: A \rightarrow B$ such that for all morphisms $f: A \rightarrow C$ and $g: B \rightarrow C$, the following diagram commutes:

$$ A @>{\phi}>> B \\ @V{f}VV @VV{g}V \\ C @>>> C \end{CD}$$ In particular, if $F$ is an endofunctor of a category $\mathcal{C}$ and $X$ is an object in $\mathcal{C}$, we say that the functor $F$ is **transportable** to itself on the object $X$ if there exists a natural transformation $\eta: F \rightarrow 1_X$ such that for all morphisms $f: X \rightarrow Y$, the following diagram commutes: $$

F(X) @>{\eta_X}>> 1_{F(X)} \\ @V{F(f)}VV @VV{\mathrm{id}_{F(Y)}}V \\ F(Y) @>>> F(Y) \end{CD}$$

Worked Example

Consider the category $\mathbf{R}$ of real numbers with addition as the operation. Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be defined by $f(x) = 2x$. We claim that the functor $F$ given by $F(X) = X + 1$ is transportable to itself on $\mathbb{R}$.

To show this, we need to define a natural transformation $\eta: F \rightarrow 1_{\mathbb{R}}$. Let $\eta_X: F(\mathbb{R}) \rightarrow 1_{F(\mathbb{R})}$ be given by $\eta_X(x) = x - 1$ for all $x \in \mathbb{R}$. It is easy to check that the diagram commutes:

$$ F(\mathbb{R}) @>{\eta_X}>> 1_{F(\mathbb{R})} \\ @V{F(f)}VV @VV{\mathrm{id}_{F(\mathbb{R})}}V \\ F(\mathbb{R}) @>>> F(\mathbb{R}) \end{CD}$$ Therefore, the functor $F$ is transportable to itself on $\mathbb{R}$. ### Key Theorem If $F$ is a transportable endofunctor of a category $\mathcal{C}$ and $\eta: F \rightarrow 1_C$ is a natural transformation on an object $X$, then the functor $F$ preserves limits.
