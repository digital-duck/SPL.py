Transport

Transport is a fundamental concept in mathematics, particularly in calculus and analysis. It refers to the process of moving or transforming a quantity from one place to another while preserving its essential properties.

Formally, given two metric spaces (X, d_X) and (Y, d_Y), a transport map T: X ↦ Y is a measurable function such that for every x ∈ X and y ∈ Y, we have:

d_X(x, x') = d_Y(T(x), T(x'))

This equation represents the preservation of distances under the transformation T. The set of all transport maps between two metric spaces (X, d_X) and (Y, d_Y) is denoted by C(X, Y).

Worked Example

Consider the case where X = [0, 1] ⊂ ℝ with the standard Euclidean distance d_X(x, y) = |x - y|, and Y = [0, 1] ⊂ ℝ with d_Y(u, v) = |u - v|. We want to find a transport map T: [0, 1] ↦ [0, 1] such that the following condition is satisfied:

d_X(x, x') = d_Y(T(x), T(x'))

Let's assume that T(x) = ax + (1-a)x' for some fixed x' ∈ Y. Then we can write:

d_X(x, x') = |x - x'| = |a x + (1-a)x' - a x| = |(1-a)(x-x')|

d_Y(T(x), T(x')) = |ax + (1-a)x' - ax| = |(1-a)(x'-x)|

We want to find the value of a such that:

|(1-a)(x-x')| = |(1-a)(x'-x)|

This is true for all x, x' ∈ [0, 1] if and only if a = 1/2. Therefore, the transport map T(x) = (1/2)x + (1/2)x' satisfies the required condition.

Key Theorem

The Kantorovich-Rubinstein theorem states that for any two metric spaces (X, d_X), (Y, d_Y), and any measurable function f: X ↦ Y, there exists a transport map T: X ↦ Y such that:

f(x) = min_f(y)f(T^{-1}(y))

where f^(-1) denotes the inverse of f.
