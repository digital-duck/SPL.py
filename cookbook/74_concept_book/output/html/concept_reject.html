Reject

Definition

In the context of decision theory and optimization, a reject is a value that is not selected or chosen as part of an optimal solution. In other words, a reject is an element that does not meet the criteria for optimality.

Formally, given a set of possible solutions $\mathcal{S}$ and a criterion for optimality $f: \mathcal{S} \to \mathbb{R}$, we define a reject as follows:

$$\text{reject}(\mathcal{S}, f) = \{s \in \mathcal{S} : f(s) < f^*(s)\},$$

where $f^*$ denotes the optimal value function.

Worked Example

Consider a university admissions problem where we want to select students with the highest GPAs. Let $\mathcal{S}$ be the set of all admitted students, and let $f(s)$ denote the GPA of student $s$. Suppose we have two students, Alice and Bob, with GPAs 3.5 and 3.2, respectively. We want to maximize the average GPA of our incoming class.

Let's define a function that calculates the average GPA:

$$g(\mathcal{S}) = \frac{1}{|\mathcal{S}|} \sum_{s \in \mathcal{S}} f(s).$$

We can evaluate $g$ for different subsets of students and keep track of the maximum value. Suppose we have a set $\mathcal{T}$ containing Alice, Bob, and another student Charlie with GPA 3.1. We find that:

\begin{align*} g(\mathcal{T}) &= \frac{1}{3} (f(Alice) + f(Bob) + f(Charlie)) \\ &= \frac{1}{3} (3.5 + 3.2 + 3.1) \\ &= 3.26. \end{align*}

However, if we add Charlie to the set $\mathcal{T}$ and remove Bob, we get:

$$g(\{\text{Alice}, \text{Charlie}\}) = \frac{1}{2} (f(Alice) + f(Charlie)) = 3.35.$$

In this case, removing Bob from consideration is a rejection of his application in favor of Alice's.

Key Theorem

The following theorem provides a necessary and sufficient condition for optimality:

Rejection Criterion

Given a function $f: \mathcal{S} \to \mathbb{R}$ and a criterion for optimality, $\text{reject}(\mathcal{S}, f)$ is non-empty if and only if there exists an optimal solution $s^* \in \mathcal{S}$ such that:

$$\forall s \in \mathcal{S} : f(s) < f(s^*).$$

Lab Cell (SymPy)

import sympy as sp

# Define the function for GPA
def g(S):
    return sum(sp.symbols('f' + str(i)) for i in range(len(S))) / len(S)

# Evaluate g for different subsets of students
S = ['Alice', 'Bob', 'Charlie']
T = S[:2] + ['Charlie']

print(g(T))
