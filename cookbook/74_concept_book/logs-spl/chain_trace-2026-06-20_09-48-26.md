### distance (39.4s)

## Distance

**Definition.** Let $\mathbb{R}^n$ denote $n$-dimensional Euclidean space. Given two points $\mathbf{p} = (p_1, p_2, \ldots, p_n)$ and $\mathbf{q} = (q_1, q_2, \ldots, q_n)$, the **Euclidean distance** between them is

$$d(\mathbf{p}, \mathbf{q}) = \|\mathbf{p} - \mathbf{q}\| = \sqrt{\sum_{i=1}^{n}(p_i - q_i)^2}.$$

This formula generalises the familiar Pythagorean theorem: in $\mathbb{R}^2$ it reduces to $\sqrt{(p_1-q_1)^2+(p_2-q_2)^2}$, the length of the hypotenuse of the right triangle whose legs run parallel to the coordinate axes. The function $d$ is non-negative, equals zero precisely when $\mathbf{p} = \mathbf{q}$, and is symmetric: $d(\mathbf{p},\mathbf{q}) = d(\mathbf{q},\mathbf{p})$.

**Worked Example.** Find the Euclidean distance between $A = (1, -2, 4)$ and $B = (5, 2, 1)$ in $\mathbb{R}^3$.

$$d(A, B) = \sqrt{(5-1)^2 + (2-(-2))^2 + (1-4)^2} = \sqrt{16 + 16 + 9} = \sqrt{41} \approx 6.403.$$

Note that the result is independent of the order of the arguments: $d(A,B) = d(B,A)$, consistent with symmetry.

**Key Theorem (Triangle Inequality in $\mathbb{R}^n$).** For all $\mathbf{p}, \mathbf{q}, \mathbf{r} \in \mathbb{R}^n$,

$$d(\mathbf{p}, \mathbf{r}) \leq d(\mathbf{p}, \mathbf{q}) + d(\mathbf{q}, \mathbf{r}).$$

*Proof sketch.* Set $\mathbf{u} = \mathbf{q} - \mathbf{p}$ and $\mathbf{v} = \mathbf{r} - \mathbf{q}$, so $\mathbf{r} - \mathbf{p} = \mathbf{u} + \mathbf{v}$. Then $\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$ follows directly from the Cauchy–Schwarz inequality $|\mathbf{u} \cdot \mathbf{v}| \leq \|\mathbf{u}\|\|\mathbf{v}\|$. $\square$

This inequality captures the geometric intuition that a straight-line path between two points is never longer than any detour through a third point. In later sections, this property will be seen as one of the defining axioms of the broader class of distance functions known as metrics.

**Lab Cell (SymPy).**

```python
from sympy import sqrt, Matrix

A = Matrix([1, -2, 4])
B = Matrix([5,  2, 1])

diff = B - A
dist = sqrt(diff.dot(diff))
print(f"d(A, B) = {dist} ≈ {float(dist):.4f}")
# d(A, B) = sqrt(41) ≈ 6.4031
```

Running this cell confirms the analytic result and illustrates how SymPy preserves exact radical form before converting to a decimal approximation.

### point (35.5s)

## Point

**Definition.** A *point* is the most fundamental object in geometry: it has no dimension, no length, no area, and no volume — it is a precise location in space. In the Euclidean plane $\mathbb{R}^2$, a point $P$ is an ordered pair $(x, y)$ where $x, y \in \mathbb{R}$. More generally, a point in $\mathbb{R}^n$ is an ordered $n$-tuple $(x_1, x_2, \ldots, x_n)$. Points are the atomic building blocks from which all geometric figures — lines, curves, surfaces — are constructed.

The **distance** between two points $P = (x_1, y_1)$ and $Q = (x_2, y_2)$ in $\mathbb{R}^2$ is given by the Euclidean metric:

$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.$$

This formula extends directly to $\mathbb{R}^n$: $d(P, Q) = \sqrt{\sum_{i=1}^{n}(q_i - p_i)^2}$.

**Worked Example.** Let $P = (1, 3)$ and $Q = (4, 7)$. Compute the distance $d(P, Q)$.

$$d(P, Q) = \sqrt{(4-1)^2 + (7-3)^2} = \sqrt{9 + 16} = \sqrt{25} = 5.$$

The displacement in the $x$-direction is $3$ and in the $y$-direction is $4$; these form the legs of a right triangle whose hypotenuse — the direct distance between the two points — has length $5$. This connection to the Pythagorean theorem is not coincidental: the Euclidean metric is defined precisely to encode it.

**Key Theorem (Triangle Inequality).** For any three points $A$, $B$, $C$ in $\mathbb{R}^n$,
$$d(A, C) \leq d(A, B) + d(B, C),$$
with equality if and only if $B$ lies on the line segment $\overline{AC}$. This is the cornerstone of metric geometry: no indirect path can be shorter than the direct one. A proof sketch applies the Cauchy–Schwarz inequality to the displacement vectors $\mathbf{u} = B - A$ and $\mathbf{v} = C - B$, noting that $\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$.

**Lab Cell (SymPy).**

```python
from sympy import Point

P = Point(1, 3)
Q = Point(4, 7)

dist = P.distance(Q)          # returns 5

print(f"Distance : {dist}")
```

Running this cell confirms $d(P, Q) = 5$, agreeing with the hand calculation above. Students are encouraged to modify the coordinates and verify that the computed distance always satisfies the Triangle Inequality for a third point $R$ of their choosing.

### circle (63.5s)

## Circle

**Definition.** A *circle* is the set of all points in the Euclidean plane that lie at a fixed distance $r > 0$ from a given point $C = (h, k)$, called the *centre*. Formally,

$$\mathcal{C} = \{(x, y) \in \mathbb{R}^2 \mid (x - h)^2 + (y - k)^2 = r^2\}.$$

The constant $r$ is the *radius*. When the centre is the origin, the equation reduces to $x^2 + y^2 = r^2$. Expanding the standard form yields the *general form*

$$x^2 + y^2 + Dx + Ey + F = 0,$$

where $D = -2h$, $E = -2k$, and $F = h^2 + k^2 - r^2$. A general second-degree equation of this type represents a circle precisely when $D^2 + E^2 - 4F > 0$.

**Worked Example.** Identify the centre and radius of $x^2 + y^2 - 6x + 4y - 3 = 0$.

*Complete the square* in each variable:

$$
(x^2 - 6x + 9) + (y^2 + 4y + 4) = 3 + 9 + 4,
$$

$$
(x - 3)^2 + (y + 2)^2 = 16.
$$

Hence the centre is $(3, -2)$ and the radius is $r = \sqrt{16} = 4$.

**Key Theorem (Tangent–Radius Perpendicularity).** *A tangent line to a circle is perpendicular to the radius drawn to the point of tangency.* Precisely, if $T$ lies on $\mathcal{C}$ and $\ell$ is the unique line through $T$ meeting $\mathcal{C}$ at exactly one point (the *tangent* at $T$), then $\ell \perp CT$.

*Proof sketch.* Suppose $\ell$ is not perpendicular to $CT$. Then the foot of the perpendicular from $C$ to $\ell$ is a point $Q \neq T$ with $|CQ| < |CT| = r$. Points of $\ell$ sufficiently close to $Q$ therefore lie strictly inside $\mathcal{C}$, forcing $\ell$ to cross $\mathcal{C}$ at a second point — contradicting the assumption that $\ell$ is tangent. $\square$

**Lab Cell (SymPy).**

```python
from sympy import symbols, sqrt

x, y = symbols('x y')

# Complete the square for x^2 + y^2 - 6x + 4y - 3 = 0
completed = (x - 3)**2 + (y + 2)**2 - 16

centre = (3, -2)
radius = sqrt(16)
print(f"Centre: {centre}, Radius: {radius}")   # Centre: (3, -2), Radius: 4

# Verify: the rightmost point lies on the circle
x_pt, y_pt = 3 + 4, -2
print(completed.subs({x: x_pt, y: y_pt}))     # 0
```
