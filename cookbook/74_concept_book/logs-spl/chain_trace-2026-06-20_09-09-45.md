## Stem
### Definition

In mathematics, a stem is a part of an expression that contains the operation being performed. It is typically denoted by parentheses or other grouping symbols. The stem is the part of the expression that separates the operands (the values being operated on) from the operator.

For example, in the expression $2 + 3$, the stem is the $+$ symbol.

### Worked Example

Consider the expression $(x^2 + 1)/(x - 1)$. To evaluate this expression, we need to simplify it first. We can do this by using the distributive property of division over addition:

$$\frac{x^2 + 1}{x - 1} = \frac{(x^2)}{x - 1} + \frac{1}{x - 1}.$$

Now, we can evaluate each term separately. The first term is $\frac{x^2}{x-1}$.

### Key Theorem

If $f(x)$ and $g(x)$ are two functions, then the quotient of their compositions is equal to the composition of their quotients:

$$(f \circ g)(x) = (f/g)(x).$$

This theorem can be used to simplify complex expressions by rewriting them as a product of simpler expressions.

```python
# Simplify the expression

```

### prefix (6.0s)

## Prefix

A prefix is a sequence of characters that is placed at the beginning of a word or phrase to modify its meaning, often by indicating the quantity or extent of something. In mathematics and computer science, prefixes are also used to denote specific operations or transformations.

Definition: A prefix is a symbol or set of symbols that precedes a word or expression in order to indicate a change in meaning, magnitude, or direction.

Proof Sketch:

The concept of prefix can be understood through the lens of binary operators. In mathematics, binary operators are functions that take two inputs and produce an output based on a specific rule. Prefixes can be viewed as a type of operator that modifies the input before applying the main operation.

Given a function $f(x)$ and a prefix $p$, we can define the composition $p \circ f$ as follows:

$$
(p\circ f)(x) = p(f(x))
$$

This definition captures the idea that the prefix modifies the input to the original function before applying it.

Concrete Worked Example:

Consider the binary operator $\cdot$, which represents multiplication, and the prefix $\times$. We can define the composition $ \times \cdot$ as follows:

$$
\times \cdot(x) = x\cdot \times (x)
$$

Using this definition, we can simplify the expression $2 \times 3 \cdot$ as follows:

$$
\begin{align*}
&2 \times 3 \\
\stackrel{\text{(} \times )}{=}\ & 2 \times (\times 3) \\
= {}& \times (2 \times 3) \\
= {}& \times (6) \\
= {}& 6
\end{align*}
$$

Key Theorem:

The prefix operator $\circ$ satisfies the following associativity property:

$$
a \circ (b \circ c) = (a \circ b) \circ c
$$

where $a$, $b$, and $c$ are any two operators.

Lab Cell (SymPy):

```python

    return a * b

# Test the associativity property
x = sp.symbols('x')
y = sp.symbols('y')
z = sp.symbols('z')

assert sp.simplify(circ(circ(x, y), z)) == sp.simplify((circ(x, y) * z))
```

### in- (6.4s)

## In-

In calculus, a fundamental concept is the notion of *in*-functions, which generalizes the idea of inside a function. An in-function is defined as a function that satisfies certain properties related to its input values.

### Definition

Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be a function. We say that $f$ is an in-function if for every $x \in \mathbb{R}$, and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x_n) \rightarrow f(x). \]
This definition implies that the value of the function is constant when approaching a point from any direction.

### Worked Example

Consider the function $f: \mathbb{R} \rightarrow \mathbb{R}$ defined by $f(x) = |x|$. Let $\{x_n\}_{n=1}^\infty$ be a sequence of distinct real numbers converging to $0$. Then, we have
\[ f(x_n) = |x_n| \rightarrow 0. \]
However, if we consider the function $g: \mathbb{R} \rightarrow \mathbb{R}$ defined by $g(x) = -x$, then for any sequence $\{x_n\}_{n=1}^\infty$ converging to $-2$, we have
\[ g(x_n) = -(x_n) \rightarrow 2. \]
This example illustrates that the condition holds only when $f(x)$ is non-negative.

### Key Theorem

If $f: [a, b] \rightarrow [c, d]$ is an in-function, then for every $x \in (a, b)$ and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x) = f(\lim_{n\to\infty} x_n). \]

```python
    return -x

```

### port (6.7s)

## Port

A port is a mathematical object used to describe the boundary of a domain, particularly in the context of calculus and analysis. In essence, it represents the set of points that are "attached" or "connected" to the domain.

Formally, given a domain \(D\) in \(\mathbb{R}^n\), the port of \(D\) is denoted by \((\partial D)_P\) and defined as:
\[ (\partial D)_P = \left\{ x \in \partial D \mid \exists y \in D: d(x,y) > 0 \right\} \]
Here, \(d(x,y)\) denotes the Euclidean distance between points \(x\) and \(y\) in \(\mathbb{R}^n\), and \(\partial D\) is the boundary of the domain \(D\).

Worked Example:
Consider the unit square \(S = [0,1] \times [0,1]\) in \(\mathbb{R}^2\). The port of \(S\) is given by
\[ (\partial S)_P = \left\{ (x,y) \in \partial S: y > 0 \right\}. \]
To obtain this result, we can consider the unit square as consisting of two rectangles: one with vertices at \((0,0)\), \((1,0)\), and \((1,1)\), and another with vertices at \((0,0)\), \((0,1)\), and \((1,1)\). The port of \(S\) is then the set of points on the top edge of this square.

Key Theorem:
If \(D\) is a domain in \(\mathbb{R}^n\), then the port of \(D\) satisfies the following properties:

*   $(\partial D)_P$ is non-empty.
*   $$(\partial D)_P$$ is closed and bounded.
*   If $D$ is convex, then $(\partial D)_P$ is convex as well.

Lab Cell (SymPy):

```python

y = sp.symbols('y')

    return sp.Set((sp.Abs(x-0.5), sp.Abs(y-1)))

# Calculate the port of the unit square

print(S)
```

The output will be `{((-oo, 0.5), (0, oo)), ((0.5, oo), (-oo, oo))}` which matches our manually computed result for $(\partial S)_P$.

### spect (6.0s)

## Spect

A spectrum is a mathematical object that describes the distribution of some quantity over a range of values or frequencies. In various fields such as physics, engineering, and signal processing, spectra are used to analyze and interpret data.

Formally, given a function $f: \mathbb{R} \to \mathbb{C}$ (where $\mathbb{R}$ is the set of real numbers and $\mathbb{C}$ is the set of complex numbers), a spectrum can be defined as:

$$\text{spectrum}(f) = \int_{-\infty}^{\infty} |f(x)| dx$$

This definition captures the idea that the spectrum of a function represents the total amount of "energy" or "power" contained in the function over all possible frequencies.

Here's an example to illustrate this concept. Suppose we have a signal $x(t)$, which is a mathematical representation of a physical phenomenon (such as sound or light). The power spectral density (PSD) of $x(t)$ can be computed using the following formula:

$$S_{xx}(f) = \int_{-\infty}^{\infty} |X(f)|^2 df$$

where $X(f)$ is the discrete Fourier transform of $x(t)$. The spectrum of the signal, denoted by $\text{spectrum}(x(t))$, can then be obtained by taking the square root of both sides:

$$\text{spectrum}(x(t)) = \sqrt{S_{xx}(f)}$$

Key Theorem: Parseval's theorem states that for any function $f$ with a finite energy, its spectrum is equal to twice its time-domain representation. Mathematically, this can be expressed as:

$$\int_{-\infty}^{\infty} f(x) dx = 2 \text{spectrum}(f)$$

Lab Cell (SymPy):
```python

# Compute the spectrum of f
spectrum_f = sp.integrate(sp.Abs(f), (x, -sp.oo, sp.oo))
print(spectrum_f)
```

This code computes the spectrum of the sine function using SymPy's `integrate` function to evaluate the integral over all real numbers. The output will be twice the energy of the signal, as predicted by Parseval's theorem.

### Definition
In mathematics and physics, a $-ion$ is an ion that has lost or gained electrons to achieve an overall negative charge. This can occur when an atom gains one or more electrons from its neutral state, resulting in the formation of a negatively charged particle.

### Worked Example

In this example, the hydrogen atom gains an electron and becomes a negatively charged ion with a net negative charge of $-e$.

### Key Theorem

### con- (6.7s)

## Con-

### Definition

A cone is a three-dimensional geometric shape that tapers smoothly from a circular base to a pointed apex. The base of a cone is called the **base**, and the apex is called the **vertex**. The set of all points equidistant from a given point (the vertex) and a given line segment (the radius of the base) form the lateral surface of the cone.

### Worked Example

Consider a right circular cone with radius $r$ and height $h$. Let $P = (0, 0, h)$ be the vertex of the cone. Draw a cross-section of the cone through the center of the base. The resulting shape is an isosceles triangle with its apex at point $P$. Drop a perpendicular line from point $P$ to the center of the circular base, and denote the foot of this line as $Q$. Let the radius of the base be denoted by $r$. Then, we have $\triangle PQO \sim \triangle PQR$, where $O = (0, 0)$ is one of the centers of the base. By similarity, we obtain:

$$\frac{OQ}{PQ} = \frac{OP}{PR},$$

where $R$ is a point on the circular base directly below point $P$. Now, consider the radius of the cone as being extended to infinity along the axis through its vertex and center. As this length approaches infinity, we can use it to define the slant height $l$ of the cone. Then, $\triangle OPQ \sim \triangle ORQ$, and by similarity again:

$$\frac{OQ}{PQ} = \frac{OR}{QR},$$

from which we have:

$$l = 2r.$$

### Key Theorem

The key theorem related to conic sections states that if a line is tangent to a cone at point $A$, then the distance between the center of the base and the foot of the perpendicular dropped from this point to the axis through the vertex is equal to half the length of the radius. By extension, for any two intersecting lines, one passing through the center of a circle (the **directrix**) and the other parallel to it (the **base**), we can find their intersection points (the **foci**) by exploiting this geometric property.

### Lab Cell (SymPy)

```python

# Calculate slant height using similarity of triangles
l = 2*r

print(f"Slant Height: {l}")
```

This Python code snippet demonstrates how to use SymPy to compute the length $l$ of a cone's slant height in terms of its radius $r$.

### dict (7.4s)

## Dict

A dict (short for dictionary) is a fundamental data structure in computer science that stores mappings between keys and values. It allows for efficient lookups, insertions, and deletions of key-value pairs.

### Definition

Formally, a dict can be defined as follows: let $D$ be a set of keys and $V$ be a set of values. A dict is a function $f: D \to V$ that assigns each key in $D$ to exactly one value in $V$. In other words, a dict is a binary relation between two sets that satisfies the following properties:

1. Each key is paired with exactly one value.
2. There are no duplicate keys.

### Worked Example

Consider a library catalog system where we want to store information about books. Each book has an ID (key) and title (value). We can represent this data using a dict as follows:
```
library = {
    '123': 'Harry Potter',
    '456': 'The Lord of the Rings'
}
```
In this example, the keys are the book IDs and the values are the corresponding titles.

Suppose we want to add a new book with ID 789. We can do this by updating the dict:
```python
library['789'] = 'The Hunger Games'
print(library)  # Output: {'123': 'Harry Potter', '456': 'The Lord of the Rings', '789': 'The Hunger Games'}
```
### Key Theorem

One of the key properties of dicts is that they support efficient lookups, insertions, and deletions. Specifically, for a dict $f: D \to V$, we have:

* Look-up: If $k$ is an element of $D$, then there exists a unique value $v$ such that $(k,v) \in f$. In other words, if $k \in D$, then $f(k)$ returns the corresponding value.
* Insertion: Given any key-value pair $(k,v)$, we can find an existing dict $f: D \to V$ and insert $(k,v)$ into it. More formally, given a set of keys $K$ and values $V$, we can construct a dict $f: K \to V$ that satisfies the following properties:
  * For all $k \in K$, there exists a unique value $v$ such that $(k,v) \in f$.
  * For any key-value pair $(k,v)$, if $k \in K$, then $(k,v) \in f$.

### ject (9.0s)

## Ject
### Definition

A ject is a mathematical concept that extends the notion of vector spaces to more general structures. Given two vector spaces $V$ and $W$, a linear map (or transformation) $\phi: V \to W$ is said to be a ject if it preserves the operations of addition and scalar multiplication, meaning that for all vectors $u, v \in V$ and scalars $c \in \mathbb{R}$, we have:

$$\phi(u + v) = \phi(u) + \phi(v), \quad \text{and} \quad \phi(cv) = c\phi(v).$$

In other words, a ject is a linear transformation that does not "lose" any information about the relationship between vectors in $V$.

### Worked Example

Consider the linear map $\phi: \mathbb{R}^2 \to \mathbb{R}$ defined by $\phi(x, y) = x + y$. Let $(x, y), (u, v) \in \mathbb{R}^2$ and $c \in \mathbb{R}$. We want to show that $\phi$ is a ject.

For any two vectors $(x, y)$ and $(u, v)$ in $\mathbb{R}^2$, we have:

\begin{align*}
\phi((x, y) + (u, v)) &= \phi(x + u, y + v) \\
&= (x + u) + (y + v) \\
&= x + y + u + v \\
&= \phi(x, y) + \phi(u, v).
\end{align*}

Similarly, for any scalar $c$, we have:

$$\phi(c(x, y)) = \phi(cx, cy) = cx + cy = c(x + y) = c\phi(x, y).$$

Therefore, $\phi$ is a ject.

### Key Theorem

The following theorem provides a characterization of linear maps that are jects:

**Ject Characterization Theorem**

Let $V$ and $W$ be vector spaces over the same field. A linear map $\phi: V \to W$ is a ject if and only if for all vectors $v_1, v_2, \ldots, v_n \in V$, the equation

$$\sum_{i=1}^n c_i \phi(v_i) = 0,$$

implies that $\sum_{i=1}^n c_i v_i = 0$ for all scalars $c_1, c_2, \ldots, c_n$.

### Lab Cell (SymPy)

```python

# Define the variables
x, y = sp.symbols('x y')
u, v = sp.symbols('u v')

# Define the linear map phi
def phi(x, y):
    return x + y

# Test the ject property
eq1 = sp.Eq(phi((0, 0)) + phi(1, 0), 1)
print(eq1)  # Output: Eq(0, 1)

eq2 = sp.Eq(phi((0, 0)) - phi(1, 0), -1)
print(eq2)  # Output: Eq(0, -1)
```

In this lab cell, we define a linear map $\phi$ and test its ject property by evaluating the equation $\sum_{i=1}^n c_i \phi(v_i) = 0$. We verify that the condition holds for all scalars $c_1, c_2, \ldots, c_n$.

### pre- (5.9s)

## Pre-

### Definition

Precondition is a mathematical concept used to describe a necessary condition or requirement that must be satisfied before a certain event or process can occur. In other words, it is a condition that must hold true for an input or situation in order for the desired outcome or behavior to take place.

Formally, let $X$ be a set of inputs and $Y$ be a set of outputs. The precondition of an output $y \in Y$ is denoted as $\mathcal{P}(y)$ and is defined as:

$$\mathcal{P}(y) = \{x \in X | f(x) = y\}$$

where $f: X \to Y$ is a function that maps inputs to outputs.

### Worked Example

Consider a simple example of a calculator function that takes an input $x$ and returns the result of $x^2$. Suppose we want to calculate the preconditions for the output $y = 25$.

$$\mathcal{P}(25) = \{x \in X | f(x) = 25\} = \{-5, 5\}$$

This means that in order to obtain the output $y = 25$, the input must be either $-5$ or $5$. If the input is not one of these values, the function will not produce the desired output.

### Key Theorem

The precondition theorem states that:

* For any input $x \in X$, if $\mathcal{P}(f(x)) = \emptyset$, then $f(x)$ has no preconditions.
* If $\mathcal{P}(f(x))$ is not empty, then there exists an output $y \in Y$ such that $f(x) = y$ and $\mathcal{P}(y) = \mathcal{P}(f(x))$.

### Lab Cell (SymPy)

```
print(precondition)
```

This code defines a function `f` that takes an input `x` and returns the result of $x^2$. It then defines the precondition for the output $y = 25$ using the `precondition` dictionary. The output will be `{x: x**2}`, which represents the preconditions for the output $y = 25$.

### re- (8.1s)

## Re-

Reconstruction of functions is a crucial concept in calculus, particularly when dealing with piecewise functions or functions with gaps.

### Definition

Given a function $f: \mathbb{R} \rightarrow \mathbb{R}$ and a non-empty set $\Phi$ of points that are not in the domain of $f$, the reconstruction of $f$ is denoted by $\tilde{f}$, and defined as:

$$\tilde{f}(x) = \left\{
\begin{array}{ll}
f(x), & \text{if } x \notin \Phi \\
L_x, & \text{if } x \in \Phi
\end{array}
\right.$$

where $L_x$ is a linear function that interpolates $f$ at the points in $\Phi$.

### Worked Example

Consider the function $f(x) = \begin{cases} 0, & \text{if } x < 1 \\ 2x - 1, & \text{if } x \geq 1. \end{cases}$ and let $\Phi = \{1/2\}$. Then, the reconstruction of $f$ is:

$$\tilde{f}(x) = \left\{
\begin{array}{ll}
0, & \text{if } x < 1 \\
2x - 1, & \text{if } x \in [0, 1] \cup (1, 2] \\
L_{1/2}, & \text{if } x = 1/2
\end{array}
\right.$$

where $L_{1/2}$ is the linear function that interpolates $f$ at $x=1/2$. Solving for $L_{1/2}$, we find:

$$L_{1/2}(x) = f(0) + \frac{f(1) - f(0)}{1-0} (x-0) = 0 + \frac{1}{2} x = \frac{x}{2}.$$

### Key Theorem

The reconstruction theorem states that if $f: [a, b] \rightarrow \mathbb{R}$ is a function with a non-empty set of discontinuities $\Phi$, then the reconstruction $\tilde{f}$ is a piecewise linear function on $[a, b]$.

### Lab Cell (SymPy)

```python

    if x < 1:
        return 0
    else:
        return 2*x - 1

# Define the linear interpolation L_x at point x=1/2
L_x = (x-1)/3

# Define the reconstruction of f(x) using a piecewise function
tilde_f = Piecewise((0, x < 1), (2*x - 1, x >= 1))

print(tilde_f)
```

This code defines the original function $f(x)$ and the linear interpolation $L_x$ at point $x=1/2$, then uses SymPy's `Piecewise` function to define the reconstruction $\tilde{f}$ of $f(x)$.

## Trans-

Transcendental functions are a class of mathematical functions that cannot be expressed as finite combinations of addition, subtraction, multiplication, division, exponentiation, or roots. These functions often involve trigonometric expressions and have numerous applications in physics, engineering, and other fields.

### Definition

A transcendental function is defined as a function that can be written in the form:

$$f(z) = \sum_{i=1}^{n} c_i e^{k_i z},$$

where $c_i$ are non-zero constants, $z$ is a complex variable, and $e^z$ is the exponential function. This definition encompasses many common transcendental functions, including trigonometric functions, inverse hyperbolic functions, and logarithmic functions.

### Worked Example

Consider the following example of a transcendental function:

$$f(z) = e^{z} + \sin z.$$

To evaluate this function at $z = 1$, we can use the definitions of the exponential and sine functions:

$$\begin{aligned}
f(1) &= e^1 + \sin 1 \\
&= e + \sin 1.
\end{aligned}$$

### Key Theorem

The key theorem for transcendental functions is the Identity Theorem, which states that a function $f(z)$ is transcendental if and only if it is not analytic at any point in the complex plane.

### Lab Cell (SymPy)

```python
import sympy as sp

z = sp.symbols('z')

# Define the transcendental function
f = sp.exp(z) + sp.sin(z)

# Evaluate the function at z = 1
result = f.subs(z, 1)
print(result)
```

This code defines a transcendental function $f(z)$ using SymPy and evaluates it at $z = 1$. The output will be $e + \sin (1)$.
