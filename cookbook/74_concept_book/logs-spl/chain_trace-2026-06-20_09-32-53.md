## Stem
### Definition

In mathematics, a stem is a part of an expression that contains the operation being performed. It is typically denoted by parentheses or other grouping symbols. The stem is the part of the expression that separates the operands (the values being operated on) from the operator.

For example, in the expression $2 + 3$, the stem is the $+$ symbol.

### Worked Example

Consider the expression $(x^2 + 1)/(x - 1)$. To evaluate this expression, we need to simplify it first. We can do this by using the distributive property of division over addition:

$$\frac{x^2 + 1}{x - 1} = \frac{(x^2)}{x - 1} + \frac{1}{x - 1}.$$

Now, we can evaluate each term separately. The first term is $\frac{x^2}{x-1}$.

### Key Theorem

If $f(x)$ and $g(x)$ are two functions, then the quotient of their compositions is equal to the composition of their quotients:

$$(f \circ g)(x) = (f/g)(x).$$

This theorem can be used to simplify complex expressions by rewriting them as a product of simpler expressions.


## Prefix

A prefix is a sequence of characters that is placed at the beginning of a word or phrase to modify its meaning, often by indicating the quantity or extent of something. In mathematics and computer science, prefixes are also used to denote specific operations or transformations.

Definition: A prefix is a symbol or set of symbols that precedes a word or expression in order to indicate a change in meaning, magnitude, or direction.

Proof Sketch:

The concept of prefix can be understood through the lens of binary operators. In mathematics, binary operators are functions that take two inputs and produce an output based on a specific rule. Prefixes can be viewed as a type of operator that modifies the input before applying the main operation.

Given a function $f(x)$ and a prefix $p$, we can define the composition $p \circ f$ as follows:

$$
(p\circ f)(x) = p(f(x))
$$

This definition captures the idea that the prefix modifies the input to the original function before applying it.

Concrete Worked Example:

Consider the binary operator $\cdot$, which represents multiplication, and the prefix $\times$. We can define the composition $ \times \cdot$ as follows:

$$
\times \cdot(x) = x\cdot \times (x)
$$

Using this definition, we can simplify the expression $2 \times 3 \cdot$ as follows:

$$
\begin{align*}
&2 \times 3 \\
\stackrel{\text{(} \times )}{=}\ & 2 \times (\times 3) \\
= {}& \times (2 \times 3) \\
= {}& \times (6) \\
= {}& 6
\end{align*}
$$

Key Theorem:

The prefix operator $\circ$ satisfies the following associativity property:

$$
a \circ (b \circ c) = (a \circ b) \circ c
$$

where $a$, $b$, and $c$ are any two operators.

Lab Cell (SymPy):

```python
    return a * b

# Test the associativity property
```

## In-

In calculus, a fundamental concept is the notion of *in*-functions, which generalizes the idea of inside a function. An in-function is defined as a function that satisfies certain properties related to its input values.

### Definition

Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be a function. We say that $f$ is an in-function if for every $x \in \mathbb{R}$, and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x_n) \rightarrow f(x). \]
This definition implies that the value of the function is constant when approaching a point from any direction.

### Worked Example

Consider the function $f: \mathbb{R} \rightarrow \mathbb{R}$ defined by $f(x) = |x|$. Let $\{x_n\}_{n=1}^\infty$ be a sequence of distinct real numbers converging to $0$. Then, we have
\[ f(x_n) = |x_n| \rightarrow 0. \]
However, if we consider the function $g: \mathbb{R} \rightarrow \mathbb{R}$ defined by $g(x) = -x$, then for any sequence $\{x_n\}_{n=1}^\infty$ converging to $-2$, we have
\[ g(x_n) = -(x_n) \rightarrow 2. \]
This example illustrates that the condition holds only when $f(x)$ is non-negative.

### Key Theorem

If $f: [a, b] \rightarrow [c, d]$ is an in-function, then for every $x \in (a, b)$ and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x) = f(\lim_{n\to\infty} x_n). \]


## Port

A port is a mathematical object used to describe the boundary of a domain, particularly in the context of calculus and analysis. In essence, it represents the set of points that are "attached" or "connected" to the domain.

Formally, given a domain \(D\) in \(\mathbb{R}^n\), the port of \(D\) is denoted by \((\partial D)_P\) and defined as:
\[ (\partial D)_P = \left\{ x \in \partial D \mid \exists y \in D: d(x,y) > 0 \right\} \]
Here, \(d(x,y)\) denotes the Euclidean distance between points \(x\) and \(y\) in \(\mathbb{R}^n\), and \(\partial D\) is the boundary of the domain \(D\).

Worked Example:
Consider the unit square \(S = [0,1] \times [0,1]\) in \(\mathbb{R}^2\). The port of \(S\) is given by
\[ (\partial S)_P = \left\{ (x,y) \in \partial S: y > 0 \right\}. \]
To obtain this result, we can consider the unit square as consisting of two rectangles: one with vertices at \((0,0)\), \((1,0)\), and \((1,1)\), and another with vertices at \((0,0)\), \((0,1)\), and \((1,1)\). The port of \(S\) is then the set of points on the top edge of this square.

Key Theorem:
If \(D\) is a domain in \(\mathbb{R}^n\), then the port of \(D\) satisfies the following properties:

*   $(\partial D)_P$ is non-empty.
*   $$(\partial D)_P$$ is closed and bounded.
*   If $D$ is convex, then $(\partial D)_P$ is convex as well.

Lab Cell (SymPy):

```python

    return sp.Set((sp.Abs(x-0.5), sp.Abs(y-1)))

```

## Spect

A spectrum is a mathematical object that describes the distribution of some quantity over a range of values or frequencies. In various fields such as physics, engineering, and signal processing, spectra are used to analyze and interpret data.

Formally, given a function $f: \mathbb{R} \to \mathbb{C}$ (where $\mathbb{R}$ is the set of real numbers and $\mathbb{C}$ is the set of complex numbers), a spectrum can be defined as:

$$\text{spectrum}(f) = \int_{-\infty}^{\infty} |f(x)| dx$$

This definition captures the idea that the spectrum of a function represents the total amount of "energy" or "power" contained in the function over all possible frequencies.

Here's an example to illustrate this concept. Suppose we have a signal $x(t)$, which is a mathematical representation of a physical phenomenon (such as sound or light). The power spectral density (PSD) of $x(t)$ can be computed using the following formula:

$$S_{xx}(f) = \int_{-\infty}^{\infty} |X(f)|^2 df$$

where $X(f)$ is the discrete Fourier transform of $x(t)$. The spectrum of the signal, denoted by $\text{spectrum}(x(t))$, can then be obtained by taking the square root of both sides:

$$\text{spectrum}(x(t)) = \sqrt{S_{xx}(f)}$$

Key Theorem: Parseval's theorem states that for any function $f$ with a finite energy, its spectrum is equal to twice its time-domain representation. Mathematically, this can be expressed as:

$$\int_{-\infty}^{\infty} f(x) dx = 2 \text{spectrum}(f)$$

Lab Cell (SymPy):

```python
# Compute the spectrum of f

print(spectrum_f)
```

This code computes the spectrum of the sine function using SymPy's `integrate` function to evaluate the integral over all real numbers. The output will be twice the energy of the signal, as predicted by Parseval's theorem.

## -Ion
### Definition
In mathematics and physics, a $-ion$ is an ion that has lost or gained electrons to achieve an overall negative charge. This can occur when an atom gains one or more electrons from its neutral state, resulting in the formation of a negatively charged particle.

### Worked Example
Consider a hydrogen atom with one proton and no electrons (i.e., $H^0$). If it absorbs an electron from another source, the resulting ion will have a net negative charge. Mathematically, this can be represented as:

$$\frac{1}{2} \cdot e - 1e^{-} = -e$$

where $e$ is the elementary charge ($1.6021766208 \times 10^{-19}$ C).

In this example, the hydrogen atom gains an electron and becomes a negatively charged ion with a net negative charge of $-e$.

### Key Theorem
If an atom or molecule gains or loses electrons to achieve a particular charge, it will form an ion with that charge. Conversely, if an ion releases or gains electrons, its charge will change accordingly.

## Con-

### Definition

A cone is a three-dimensional geometric shape that tapers smoothly from a circular base to a pointed apex. The base of a cone is called the **base**, and the apex is called the **vertex**. The set of all points equidistant from a given point (the vertex) and a given line segment (the radius of the base) form the lateral surface of the cone.

### Worked Example

Consider a right circular cone with radius $r$ and height $h$. Let $P = (0, 0, h)$ be the vertex of the cone. Draw a cross-section of the cone through the center of the base. The resulting shape is an isosceles triangle with its apex at point $P$. Drop a perpendicular line from point $P$ to the center of the circular base, and denote the foot of this line as $Q$. Let the radius of the base be denoted by $r$. Then, we have $\triangle PQO \sim \triangle PQR$, where $O = (0, 0)$ is one of the centers of the base. By similarity, we obtain:

$$\frac{OQ}{PQ} = \frac{OP}{PR},$$

where $R$ is a point on the circular base directly below point $P$. Now, consider the radius of the cone as being extended to infinity along the axis through its vertex and center. As this length approaches infinity, we can use it to define the slant height $l$ of the cone. Then, $\triangle OPQ \sim \triangle ORQ$, and by similarity again:

$$\frac{OQ}{PQ} = \frac{OR}{QR},$$

from which we have:

$$l = 2r.$$

### Key Theorem

The key theorem related to conic sections states that if a line is tangent to a cone at point $A$, then the distance between the center of the base and the foot of the perpendicular dropped from this point to the axis through the vertex is equal to half the length of the radius. By extension, for any two intersecting lines, one passing through the center of a circle (the **directrix**) and the other parallel to it (the **base**), we can find their intersection points (the **foci**) by exploiting this geometric property.

```python
# Calculate slant height using similarity of triangles

print(f"Slant Height: {l}")
```

This Python code snippet demonstrates how to use SymPy to compute the length $l$ of a cone's slant height in terms of its radius $r$.

## Dict

A dict (short for dictionary) is a fundamental data structure in computer science that stores mappings between keys and values. It allows for efficient lookups, insertions, and deletions of key-value pairs.

### Definition

Formally, a dict can be defined as follows: let $D$ be a set of keys and $V$ be a set of values. A dict is a function $f: D \to V$ that assigns each key in $D$ to exactly one value in $V$. In other words, a dict is a binary relation between two sets that satisfies the following properties:

1. Each key is paired with exactly one value.
2. There are no duplicate keys.

### Worked Example

Consider a library catalog system where we want to store information about books. Each book has an ID (key) and title (value). We can represent this data using a dict as follows:
```
library = {
    '123': 'Harry Potter',
    '456': 'The Lord of the Rings'
}
```
In this example, the keys are the book IDs and the values are the corresponding titles.

Suppose we want to add a new book with ID 789. We can do this by updating the dict:
```python
library['789'] = 'The Hunger Games'
print(library)  # Output: {'123': 'Harry Potter', '456': 'The Lord of the Rings', '789': 'The Hunger Games'}
```
### Key Theorem

One of the key properties of dicts is that they support efficient lookups, insertions, and deletions. Specifically, for a dict $f: D \to V$, we have:

* Look-up: If $k$ is an element of $D$, then there exists a unique value $v$ such that $(k,v) \in f$. In other words, if $k \in D$, then $f(k)$ returns the corresponding value.
* Insertion: Given any key-value pair $(k,v)$, we can find an existing dict $f: D \to V$ and insert $(k,v)$ into it. More formally, given a set of keys $K$ and values $V$, we can construct a dict $f: K \to V$ that satisfies the following properties:
  * For all $k \in K$, there exists a unique value $v$ such that $(k,v) \in f$.
  * For any key-value pair $(k,v)$, if $k \in K$, then $(k,v) \in f$.


### ject (0.0s)

## Ject
### Definition

A ject is a mathematical concept that extends the notion of vector spaces to more general structures. Given two vector spaces $V$ and $W$, a linear map (or transformation) $\phi: V \to W$ is said to be a ject if it preserves the operations of addition and scalar multiplication, meaning that for all vectors $u, v \in V$ and scalars $c \in \mathbb{R}$, we have:

$$\phi(u + v) = \phi(u) + \phi(v), \quad \text{and} \quad \phi(cv) = c\phi(v).$$

In other words, a ject is a linear transformation that does not "lose" any information about the relationship between vectors in $V$.

### Worked Example

Consider the linear map $\phi: \mathbb{R}^2 \to \mathbb{R}$ defined by $\phi(x, y) = x + y$. Let $(x, y), (u, v) \in \mathbb{R}^2$ and $c \in \mathbb{R}$. We want to show that $\phi$ is a ject.

For any two vectors $(x, y)$ and $(u, v)$ in $\mathbb{R}^2$, we have:

\begin{align*}
\phi((x, y) + (u, v)) &= \phi(x + u, y + v) \\
&= (x + u) + (y + v) \\
&= x + y + u + v \\
&= \phi(x, y) + \phi(u, v).
\end{align*}

Similarly, for any scalar $c$, we have:

$$\phi(c(x, y)) = \phi(cx, cy) = cx + cy = c(x + y) = c\phi(x, y).$$

Therefore, $\phi$ is a ject.

### Key Theorem

The following theorem provides a characterization of linear maps that are jects:

**Ject Characterization Theorem**

Let $V$ and $W$ be vector spaces over the same field. A linear map $\phi: V \to W$ is a ject if and only if for all vectors $v_1, v_2, \ldots, v_n \in V$, the equation

$$\sum_{i=1}^n c_i \phi(v_i) = 0,$$

implies that $\sum_{i=1}^n c_i v_i = 0$ for all scalars $c_1, c_2, \ldots, c_n$.

### Lab Cell (SymPy)

```python

u, v = sp.symbols('u v')

# Define the linear map phi

    return x + y

# Test the ject property
eq1 = sp.Eq(phi((0, 0)) + phi(1, 0), 1)
print(eq1)  # Output: Eq(0, 1)

eq2 = sp.Eq(phi((0, 0)) - phi(1, 0), -1)
print(eq2)  # Output: Eq(0, -1)
```

In this lab cell, we define a linear map $\phi$ and test its ject property by evaluating the equation $\sum_{i=1}^n c_i \phi(v_i) = 0$. We verify that the condition holds for all scalars $c_1, c_2, \ldots, c_n$.

### pre- (0.0s)

## Pre-

### Definition

Precondition is a mathematical concept used to describe a necessary condition or requirement that must be satisfied before a certain event or process can occur. In other words, it is a condition that must hold true for an input or situation in order for the desired outcome or behavior to take place.

Formally, let $X$ be a set of inputs and $Y$ be a set of outputs. The precondition of an output $y \in Y$ is denoted as $\mathcal{P}(y)$ and is defined as:

$$\mathcal{P}(y) = \{x \in X | f(x) = y\}$$

where $f: X \to Y$ is a function that maps inputs to outputs.

### Worked Example

Consider a simple example of a calculator function that takes an input $x$ and returns the result of $x^2$. Suppose we want to calculate the preconditions for the output $y = 25$.

$$\mathcal{P}(25) = \{x \in X | f(x) = 25\} = \{-5, 5\}$$

This means that in order to obtain the output $y = 25$, the input must be either $-5$ or $5$. If the input is not one of these values, the function will not produce the desired output.

### Key Theorem

The precondition theorem states that:

* For any input $x \in X$, if $\mathcal{P}(f(x)) = \emptyset$, then $f(x)$ has no preconditions.
* If $\mathcal{P}(f(x))$ is not empty, then there exists an output $y \in Y$ such that $f(x) = y$ and $\mathcal{P}(y) = \mathcal{P}(f(x))$.

### Lab Cell (SymPy)

```
print(precondition)
```

This code defines a function `f` that takes an input `x` and returns the result of $x^2$. It then defines the precondition for the output $y = 25$ using the `precondition` dictionary. The output will be `{x: x**2}`, which represents the preconditions for the output $y = 25$.

### re- (0.0s)

## Re-

Reconstruction of functions is a crucial concept in calculus, particularly when dealing with piecewise functions or functions with gaps.

### Definition

Given a function $f: \mathbb{R} \rightarrow \mathbb{R}$ and a non-empty set $\Phi$ of points that are not in the domain of $f$, the reconstruction of $f$ is denoted by $\tilde{f}$, and defined as:

$$\tilde{f}(x) = \left\{
\begin{array}{ll}
f(x), & \text{if } x \notin \Phi \\
L_x, & \text{if } x \in \Phi
\end{array}
\right.$$

where $L_x$ is a linear function that interpolates $f$ at the points in $\Phi$.

### Worked Example

Consider the function $f(x) = \begin{cases} 0, & \text{if } x < 1 \\ 2x - 1, & \text{if } x \geq 1. \end{cases}$ and let $\Phi = \{1/2\}$. Then, the reconstruction of $f$ is:

$$\tilde{f}(x) = \left\{
\begin{array}{ll}
0, & \text{if } x < 1 \\
2x - 1, & \text{if } x \in [0, 1] \cup (1, 2] \\
L_{1/2}, & \text{if } x = 1/2
\end{array}
\right.$$

where $L_{1/2}$ is the linear function that interpolates $f$ at $x=1/2$. Solving for $L_{1/2}$, we find:

$$L_{1/2}(x) = f(0) + \frac{f(1) - f(0)}{1-0} (x-0) = 0 + \frac{1}{2} x = \frac{x}{2}.$$

### Key Theorem

The reconstruction theorem states that if $f: [a, b] \rightarrow \mathbb{R}$ is a function with a non-empty set of discontinuities $\Phi$, then the reconstruction $\tilde{f}$ is a piecewise linear function on $[a, b]$.

### Lab Cell (SymPy)

```python
    if x < 1:
        return 0
    else:
        return 2*x - 1

# Define the reconstruction of f(x) using a piecewise function
tilde_f = Piecewise((0, x < 1), (2*x - 1, x >= 1))

print(tilde_f)
```

This code defines the original function $f(x)$ and the linear interpolation $L_x$ at point $x=1/2$, then uses SymPy's `Piecewise` function to define the reconstruction $\tilde{f}$ of $f(x)$.

## Struct

A struct is a collection of variables of different data types that are stored together in memory. It allows for efficient access and manipulation of multiple values as if they were individual variables.

### Definition

Formally, a struct can be defined as:

```latex
struct S = { x: int; y: float; z: bool }
```

## Trans-

Transcendental functions are a class of mathematical functions that cannot be expressed as finite combinations of addition, subtraction, multiplication, division, exponentiation, or roots. These functions often involve trigonometric expressions and have numerous applications in physics, engineering, and other fields.

### Definition

A transcendental function is defined as a function that can be written in the form:

$$f(z) = \sum_{i=1}^{n} c_i e^{k_i z},$$

where $c_i$ are non-zero constants, $z$ is a complex variable, and $e^z$ is the exponential function. This definition encompasses many common transcendental functions, including trigonometric functions, inverse hyperbolic functions, and logarithmic functions.

### Worked Example

Consider the following example of a transcendental function:

$$f(z) = e^{z} + \sin z.$$

To evaluate this function at $z = 1$, we can use the definitions of the exponential and sine functions:

$$\begin{aligned}
f(1) &= e^1 + \sin 1 \\
&= e + \sin 1.
\end{aligned}$$

### Key Theorem

The key theorem for transcendental functions is the Identity Theorem, which states that a function $f(z)$ is transcendental if and only if it is not analytic at any point in the complex plane.

Construct an isomorphism between $\mathbb{Z}$ (the integers) and $\mathbb{R}$ (the real numbers). Recall that an isomorphism is a bijective function that preserves the operation. ### predict (0.0s)

## Predict
Predict in the context of mathematical modeling refers to the process of estimating or forecasting future events or behavior based on past data, trends, and patterns. In various fields such as economics, finance, epidemiology, and climate science, prediction is a crucial tool for making informed decisions, predicting outcomes, and understanding complex systems.

### Definition

Prediction can be viewed as an optimization problem where the goal is to minimize the difference between observed and predicted values. Mathematically, this can be formulated as:

$$\min_{\theta} \sum_{i=1}^{n} (y_i - f(x_i, \theta))^2$$

where $y_i$ represents the observed value at time step $i$, $x_i$ is the input or feature vector, $\theta$ is the model parameter, and $f(x_i, \theta)$ is the predicted output.

### Worked Example: Simple Linear Regression

Consider a scenario where we want to predict house prices based on the number of bedrooms. We collect data on 10 houses with their corresponding prices and bedroom counts:

| Bedrooms | Price |
| --- | --- |
| 2    | $150,000 |
| 3    | $200,000 |
| 4    | $250,000 |
| ...   | ...   |

We fit a simple linear regression model to the data:

$$y = \beta_0 + \beta_1 x$$

where $x$ is the number of bedrooms and $\beta_0$ and $\beta_1$ are model parameters.

Using ordinary least squares (OLS), we find that $\hat{\beta}_1 = 50,000$ and $\hat{\beta}_0 = -25,000$. We then use this model to predict the price for a house with 3 bedrooms:

$$\hat{y} = \hat{\beta}_0 + \hat{\beta}_1 x = -25,000 + 50,000(3) = $75,000$$

### Key Theorem: Bayes' Rule

Bayes' rule provides a framework for updating the probability of a hypothesis based on new evidence. In the context of prediction, Bayes' rule can be used to update the predictive distribution of a model as more data becomes available.

Let $\pi_0$ be the prior probability of the null hypothesis $H_0$, and let $\mathcal{D}_i$ be the observed data at time step $i$. Then, Bayes' rule states:

$$\pi_{i+1}(|H_0|) = \frac{\pi_i(|H_0|)P(D_i|H_0)}{\sum_{j=0}^J \pi_i(|H_j|)P(D_i|H_j)}$$

where $\pi_i(|H_k|)$ is the posterior probability of hypothesis $H_k$ at time step $i$, and $P(D_i|H_k)$ is the likelihood of observing data $D_i$ under hypothesis $H_k$.

## Reject
### Definition

In the context of decision theory and optimization, a reject is a value that is not selected or chosen as part of an optimal solution. In other words, a reject is an element that does not meet the criteria for optimality.

Formally, given a set of possible solutions $\mathcal{S}$ and a criterion for optimality $f: \mathcal{S} \to \mathbb{R}$, we define a reject as follows:

$$\text{reject}(\mathcal{S}, f) = \{s \in \mathcal{S} : f(s) < f^*(s)\},$$

where $f^*$ denotes the optimal value function.

### Worked Example

Consider a university admissions problem where we want to select students with the highest GPAs. Let $\mathcal{S}$ be the set of all admitted students, and let $f(s)$ denote the GPA of student $s$. Suppose we have two students, Alice and Bob, with GPAs 3.5 and 3.2, respectively. We want to maximize the average GPA of our incoming class.

Let's define a function that calculates the average GPA:

$$g(\mathcal{S}) = \frac{1}{|\mathcal{S}|} \sum_{s \in \mathcal{S}} f(s).$$

We can evaluate $g$ for different subsets of students and keep track of the maximum value. Suppose we have a set $\mathcal{T}$ containing Alice, Bob, and another student Charlie with GPA 3.1. We find that:

\begin{align*}
g(\mathcal{T}) &= \frac{1}{3} (f(Alice) + f(Bob) + f(Charlie)) \\
&= \frac{1}{3} (3.5 + 3.2 + 3.1) \\
&= 3.26.
\end{align*}

However, if we add Charlie to the set $\mathcal{T}$ and remove Bob, we get:

$$g(\{\text{Alice}, \text{Charlie}\}) = \frac{1}{2} (f(Alice) + f(Charlie)) = 3.35.$$

In this case, removing Bob from consideration is a rejection of his application in favor of Alice's.

### Key Theorem

The following theorem provides a necessary and sufficient condition for optimality:

**Rejection Criterion**

Given a function $f: \mathcal{S} \to \mathbb{R}$ and a criterion for optimality, $\text{reject}(\mathcal{S}, f)$ is non-empty if and only if there exists an optimal solution $s^* \in \mathcal{S}$ such that:

$$\forall s \in \mathcal{S} : f(s) < f(s^*).$$

## Transport

Transport is a fundamental concept in mathematics, particularly in calculus and analysis. It refers to the process of moving or transforming a quantity from one place to another while preserving its essential properties.

Formally, given two metric spaces (X, d_X) and (Y, d_Y), a transport map T: X ↦ Y is a measurable function such that for every x ∈ X and y ∈ Y, we have:

d_X(x, x') = d_Y(T(x), T(x'))

This equation represents the preservation of distances under the transformation T. The set of all transport maps between two metric spaces (X, d_X) and (Y, d_Y) is denoted by C(X, Y).

### Worked Example

Consider the case where X = [0, 1] ⊂ ℝ with the standard Euclidean distance d_X(x, y) = |x - y|, and Y = [0, 1] ⊂ ℝ with d_Y(u, v) = |u - v|. We want to find a transport map T: [0, 1] ↦ [0, 1] such that the following condition is satisfied:

d_X(x, x') = d_Y(T(x), T(x'))

Let's assume that T(x) = ax + (1-a)x' for some fixed x' ∈ Y. Then we can write:

d_X(x, x') = |x - x'| = |a x + (1-a)x' - a x| = |(1-a)(x-x')|

d_Y(T(x), T(x')) = |ax + (1-a)x' - ax| = |(1-a)(x'-x)|

We want to find the value of a such that:

|(1-a)(x-x')| = |(1-a)(x'-x)|

This is true for all x, x' ∈ [0, 1] if and only if a = 1/2. Therefore, the transport map T(x) = (1/2)x + (1/2)x' satisfies the required condition.

### Key Theorem

The Kantorovich-Rubinstein theorem states that for any two metric spaces (X, d_X), (Y, d_Y), and any measurable function f: X ↦ Y, there exists a transport map T: X ↦ Y such that:

f(x) = min_f(y)f(T^{-1}(y))

where f^(-1) denotes the inverse of f.

## -Or

The -or operator is a binary logical operation that takes two arguments, x and y, and evaluates to true if at least one of them is true, but not both. This is in contrast to the usual implication, which evaluates to false only when both arguments are true.

Mathematically, the -or operation can be represented as:

$$x \lor y = \begin{cases} x &\quad \text{if } x \text{ or } y \text{ is true} \\ y &\quad \text{if } x \text{ and } y \text{ are both false} \\ \text{undefined} &\quad \text{otherwise} \end{cases}$$

### Worked Example

Consider the expression $p \lor q$, where $p$ is true and $q$ is false. Using the definition of -or, we can evaluate this as follows:

$$p \lor q = p \quad \text{since } q \text{ is false and at least one of them is true}.$$

Therefore, $p \lor q$ evaluates to true.

### Key Theorem

The -or operation satisfies the following properties:

- **Commutativity**: $x \lor y = y \lor x$
- **Associativity**: $(x \lor y) \lor z = x \lor (y \lor z)$
- **Distributivity**: $x \lor (y \land z) = (x \lor y) \land (x \lor z)$

## Construction

Construction is a fundamental concept in mathematics that serves as the foundation for various branches of algebra, geometry, and analysis. In essence, it refers to the process of building or creating mathematical structures such as points, lines, planes, and other geometric objects from axioms.

### Definition

Given a set $A$ of geometric elements (such as points, lines, or planes), a construction is a rule that associates with each element in $A$ a new object, often referred to as the "image" or "image point" of that element. The goal of a construction is to create an equivalence relation between the original elements of $A$ and their images.

Formally, let $(A, \sim)$ be a set with a binary operation $\sim$, called the equivalence relation, such that:

1. Reflexivity: for all $a \in A$, $a \sim a$
2. Symmetry: for all $a, b \in A$, if $a \sim b$ then $b \sim a$
3. Transitivity: for all $a, b, c \in A$, if $a \sim b$ and $b \sim c$, then $a \sim c$

A construction is then defined as an equivalence relation $(A, \sim)$ that satisfies the above properties.

### Worked Example

Consider a simple example of constructing points on a line. Let $\mathbb{R}$ be the set of real numbers and define the binary operation $+\) on it. Suppose we want to construct two new points $x$ and $y$ such that $x + y = 5$. We can do this by defining the equivalence relation $(\mathbb{R}, +)$ as follows:

$$a \sim b \iff a + b = 5$$

Here, the equivalence class of an element $a$ is defined as $\{b \in \mathbb{R} | a + b = 5\}$.

### Key Theorem

The construction process can be generalized using the following theorem:

**Axiom of Construction**

Given a set $A$ and a rule $f: A \to B$, where $B$ is another set, the rule $(A, f)$ defines a construction from $A$ to $B$. This rule satisfies the properties of reflexivity, symmetry, and transitivity if and only if $f$ is injective.

## Inspector

An inspector is a mathematical concept used to represent an object that can be perceived or measured through observation or experimentation. In the context of measurement theory, an inspector is defined as a function that maps a physical quantity to its corresponding measurement.

Formally, let $f: X \to Y$ be a measurement function, where $X$ represents the set of physical quantities and $Y$ represents the set of measurements. An inspector $I: Y \to 2^Y$ is defined as a mapping that takes a measurement as input and returns a subset of possible values.

## Portable

A portable object is a mathematical concept that describes a rectangular region with a fixed orientation, allowing it to be transformed into any other rectangle while preserving its proportions.

### Definition

Formally, a rectangle $P$ of dimensions $2a \times 2b$ can be considered as a portable object if there exists a scaling transformation $\alpha_P: \mathbb{R}^2 \to \mathbb{R}^2$ such that:

$$\alpha_P(x) = (ax + c, bx + d)$$

where $(c,d)$ is the translation vector of $P$, and $a,b > 0$. This transformation satisfies the condition that it preserves the aspect ratio of $P$.

### Worked Example

Suppose we have a rectangular portrait of size $2 \times 3$ with its top-left corner at the origin. We want to transform this rectangle into a new rectangle of size $4 \times 6$ while preserving its orientation. To do so, we can apply the scaling transformation $\alpha_P(x) = (x, x)$ to each point $(x,y)$ of the original rectangle.

The resulting transformed portrait has dimensions $2 \times 3$, and it is centered at the origin with a translation vector $(1,1)$.

### Key Theorem

If we are given a rectangular region $P$ with fixed orientation but non-integer dimensions, then there exists a unique scaling transformation $\alpha_P: \mathbb{R}^2 \to \mathbb{R}^2$ that preserves its aspect ratio.

## Prediction

In calculus, prediction refers to the process of estimating or forecasting a function's behavior at an unknown point based on its known properties. This concept is crucial in various fields such as physics, economics, and finance, where understanding future events or trends is essential.

Formally, given a function $f(x)$ defined on an interval $I$, prediction can be viewed as finding an expression $\hat{f}(x)$ that approximates $f(x)$ for values of $x$ outside the interval. This can be achieved by analyzing the behavior of $f(x)$ within the interval and utilizing it to make informed predictions.

Consider a simple example: predicting the population growth rate of a city based on its past growth rates. Assume we have the following data points:

$$\begin{array}{|c|c|}
\hline
x & f(x) = \text{population} \\
\hline
0 & 100,000 \\
1 & 120,000 \\
2 & 144,000 \\
\hline
\end{array}$$

Using numerical differentiation or other methods, we can estimate the derivative of $f(x)$ at $x=1$:

$$f'(1) \approx \frac{f(2)-f(1)}{2-1} = \frac{44,000}{1} = 44,000$$

This suggests that the population is growing at a rate of approximately 4,400 per year.

Key Theorem: If $f(x)$ is continuous on an interval $I$ and differentiable on $(a,b) \subset I$, then the slope of the tangent line to $f(x)$ at $x=a$ can be approximated by:

$$\frac{f(b)-f(a)}{b-a}$$

## Rejection

Rejection is a fundamental concept in decision-making and statistics that plays a crucial role in various fields, including machine learning, data analysis, and quality control.

### Definition

In general, rejection refers to the act of refusing or dismissing something, such as an offer, a proposal, or a hypothesis. In mathematical terms, rejection can be defined as a binary outcome where one option is selected and the other is discarded. This concept is often used in statistical testing, where we reject a null hypothesis if it is deemed insufficient evidence to support it.

### Worked Example

Consider a scenario where we are trying to determine whether a new product meets our quality standards. We collect a sample of products and perform a series of tests to evaluate their performance. Suppose the test results show that 90% of the products meet our standards, while 10% do not. In this case, we might reject the null hypothesis that all products meet our standards (H0) in favor of the alternative hypothesis that some products do not meet our standards (H1). We would reject H0 if the p-value associated with the test statistic is less than a certain significance level (e.g., 0.05).

### Key Theorem

The Neyman-Pearson lemma states that when testing two hypotheses, we should always reject the null hypothesis if the likelihood ratio is greater than the critical value. Mathematically, this can be expressed as:

$$\frac{L(\theta_1)}{L(\theta_0)} > k$$

where $L(\theta)$ represents the likelihood function, $\theta_0$ and $\theta_1$ are the null and alternative hypotheses, respectively, and $k$ is the critical value.

### Lab Cell (SymPy)

```python

# Define variables
x = sp.symbols('x')
p0 = 0.5  # Probability of success under H0
p1 = 0.7  # Probability of success under H1

# Likelihood ratio function
def likelihood_ratio(p):
    return (p**2) / ((1-p)**2)

# Critical value (e.g., 0.05)
critical_value = 5

# Evaluate the likelihood ratio at p0 and p1
lr_h0 = likelihood_ratio(p0)
lr_h1 = likelihood_ratio(p1)

print("Likelihood Ratio under H0:", lr_h0)
print("Likelihood Ratio under H1:", lr_h1)

if lr_h0 > critical_value:
    print("Reject H0")
else:
    print("Fail to reject H0")
```

## Transportable
A transportable is a mathematical object that can be mapped from one structure to another while preserving its essential properties. In other words, a transportable is an equivalence class of structures that are "essentially similar" in some sense.

### Definition
Given two algebraic structures $A$ and $B$ of the same type, we say that $A$ is **transportable** to $B$ if there exists a bijection $\phi: A \rightarrow B$ such that for all morphisms $f: A \rightarrow C$ and $g: B \rightarrow C$, the following diagram commutes:

$$\begin{CD}
A @>{\phi}>> B \\
@V{f}VV @VV{g}V \\
C @>>> C
\end{CD}$$

In particular, if $F$ is an endofunctor of a category $\mathcal{C}$ and $X$ is an object in $\mathcal{C}$, we say that the functor $F$ is **transportable** to itself on the object $X$ if there exists a natural transformation $\eta: F \rightarrow 1_X$ such that for all morphisms $f: X \rightarrow Y$, the following diagram commutes:

$$\begin{CD}
F(X) @>{\eta_X}>> 1_{F(X)} \\
@V{F(f)}VV @VV{\mathrm{id}_{F(Y)}}V \\
F(Y) @>>> F(Y)
\end{CD}$$

### Worked Example
Consider the category $\mathbf{R}$ of real numbers with addition as the operation. Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be defined by $f(x) = 2x$. We claim that the functor $F$ given by $F(X) = X + 1$ is transportable to itself on $\mathbb{R}$.

To show this, we need to define a natural transformation $\eta: F \rightarrow 1_{\mathbb{R}}$. Let $\eta_X: F(\mathbb{R}) \rightarrow 1_{F(\mathbb{R})}$ be given by $\eta_X(x) = x - 1$ for all $x \in \mathbb{R}$. It is easy to check that the diagram commutes:

$$\begin{CD}
F(\mathbb{R}) @>{\eta_X}>> 1_{F(\mathbb{R})} \\
@V{F(f)}VV @VV{\mathrm{id}_{F(\mathbb{R})}}V \\
F(\mathbb{R}) @>>> F(\mathbb{R})
\end{CD}$$

Therefore, the functor $F$ is transportable to itself on $\mathbb{R}$.

### Key Theorem
If $F$ is a transportable endofunctor of a category $\mathcal{C}$ and $\eta: F \rightarrow 1_C$ is a natural transformation on an object $X$, then the functor $F$ preserves limits.

Proof: Let $D$ be a limit of a diagram in $\mathcal{C}$. We need to show that $F(D)$ is also a limit. Since $\eta$ is a natural transformation, we have a commutative diagram:

$$\begin{CD}
F(X) @>{\mathrm{id}}>> F(X) \\
@V{\alpha}VV @VV{\beta}V \\
D @>>> D
\end{CD}$$

where $(X, \alpha)$ and $(Y, \beta)$ are the limit cones of the diagram over $X$ and $Y$, respectively. Since $\eta_X: F(X) \rightarrow 1_{F(X)}$ is a natural transformation, we have that:

$$\begin{CD}
F(D) @>{\mathrm{id}}>> F(D) \\
@V{\beta}VV @VV{\gamma}V \\
D @>>> D
\end{CD}$$

where $(D, \gamma)$ is the cone over $D$. Therefore, $\gamma = \beta$, and thus $F(D)$ is a limit.

## Prefix Root Composition

Prefix root composition is a fundamental operation in algebraic manipulations that allows for the simplification of expressions by combining two operations: prefixing and rooting.

### Definition

Given an expression \(f(x)\) and a polynomial \(g(x)\), the prefix root composition of \(f(x)\) with respect to \(g(x)\) is defined as:
\[ \text{pr}(f(x), g(x)) = f(g(x)) \]

In other words, we first apply the root operation to the input of function \(f\), and then apply function \(f\) to the result. This operation combines the concepts of function composition and radical expressions.

### Worked Example

Consider the expression \(f(x) = x^2 + 3x - 4\) and the polynomial \(g(x) = x - 1\). We can perform prefix root composition as follows:

\[ \text{pr}(f(x), g(x)) = f(g(x)) = (g(x))^2 + 3(g(x)) - 4 \]

Substituting \(g(x)\), we get:
\[ \text{pr}(f(x), g(x)) = ((x-1)^2) + 3((x-1)) - 4 \]
\[ \text{pr}(f(x), g(x)) = (x^2 - 2x + 1) + 3x - 3 - 4 \]
\[ \text{pr}(f(x), g(x)) = x^2 + x - 6 \]

### Key Theorem

The prefix root composition is a one-to-one correspondence between the set of all expressions of \(f(g(x))\) and the set of all expressions of the form \(a(b(c))^n\), where \(a, b, c, n \in \mathbb{R}\).

## Suffix Derivation

Suffix derivation is a fundamental concept in mathematical analysis that enables us to construct new functions from existing ones by applying specific rules. In this context, we will focus on the suffix derivation technique used to generate derivatives of functions.

### Definition

Given a function $f(x)$ and a polynomial $p(x) = \sum_{i=0}^{n} c_i x^i$, the suffix derivative of $f(x)$ with respect to $p(x)$ is denoted as $\frac{d}{dx}[f(x)p(x)]$. It can be computed using the product rule for differentiation.

### Worked Example

Consider the function $f(x) = e^x$ and the polynomial $p(x) = 3x + 2$. We want to find the suffix derivative of $f(x)$ with respect to $p(x)$.

We can write the expression $\frac{d}{dx}[f(x)p(x)]$ using the product rule as:
\begin{align*}
\frac{d}{dx}[e^x(3x+2)] &= \left(\frac{d}{dx}f(x)\right)(3x+2) + f(x) \left(\frac{d}{dx}(3x+2)\right) \\
&= e^x (3x+2) + e^x (3) \\
&= e^x(3x+5)
\end{align*}

### Key Theorem

The suffix derivative of a function $f(x)$ with respect to a polynomial $p(x) = \sum_{i=0}^{n} c_i x^i$ is given by the formula:
$$
\frac{d}{dx}[f(x)p(x)] = \left(\frac{df(x)}{dx}\right)(p(x)) + f(x) \left(\frac{dp(x)}{dx}\right)
$$

## Morphological Decoding Principle

The morphological decoding principle is a fundamental concept in natural language processing (NLP) that enables efficient decoding of morphologically rich languages such as German, Dutch, and Swedish. It is based on the observation that many words in these languages can be formed by combining smaller morphemes.

Formally, let $M$ denote the set of morphemes in a word, and let $\sigma$ be a sequence of morphemes that forms a valid word. Then, according to the morphological decoding principle, there exists an injective mapping $\phi: M \rightarrow N$, where $N$ is the set of available morphemes, such that for each $\sigma \in \Sigma$, there exists $\tau \in N^*$ with $\phi(\sigma) = \tau$.

Mathematically, this can be represented as:

$$\forall \sigma \in \Sigma, \exists \tau \in N^*, \phi(\sigma) = \tau, |\phi|_M = 1$$

where $|\cdot|$ denotes the cardinality of a mapping.

Worked Example
Consider the German word "Haus" (house). It can be decomposed into two morphemes: "Ha-" and "-us". The morphological decoding principle states that there exists an injective mapping $\phi: \{Ha-, -us\} \rightarrow \{\text{Ha}, \text{us}\}$ such that $\phi(\text{Ha-}) = \text{Ha}$ and $\phi(-\text{us}) = \text{us}$. This means that the word "Haus" can be formed by combining the morphemes "Ha-" and "-us".

Key Theorem
The morphological decoding principle is closely related to the concept of morpheme-based segmentation. Given a sequence of characters $\sigma$, the morphological decoding principle provides a way to decompose it into smaller morphemes, which can then be used for further processing such as part-of-speech tagging or named entity recognition.
