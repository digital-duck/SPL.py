## Prefix

A prefix is a sequence of characters that is placed at the beginning of a word or phrase to modify its meaning, often by indicating the quantity or extent of something. In mathematics and computer science, prefixes are also used to denote specific operations or transformations.

Definition: A prefix is a symbol or set of symbols that precedes a word or expression in order to indicate a change in meaning, magnitude, or direction.

Proof Sketch:

The concept of prefix can be understood through the lens of binary operators. In mathematics, binary operators are functions that take two inputs and produce an output based on a specific rule. Prefixes can be viewed as a type of operator that modifies the input before applying the main operation.

Given a function $f(x)$ and a prefix $p$, we can define the composition $p \circ f$ as follows:

$$
(p\circ f)(x) = p(f(x))
$$

This definition captures the idea that the prefix modifies the input to the original function before applying it.

Concrete Worked Example:

Consider the binary operator $\cdot$, which represents multiplication, and the prefix $\times$. We can define the composition $ \times \cdot$ as follows:

$$
\times \cdot(x) = x\cdot \times (x)
$$

Using this definition, we can simplify the expression $2 \times 3 \cdot$ as follows:

$$
\begin{align*}
&2 \times 3 \\
\stackrel{\text{(} \times )}{=}\ & 2 \times (\times 3) \\
= {}& \times (2 \times 3) \\
= {}& \times (6) \\
= {}& 6
\end{align*}
$$

Key Theorem:

The prefix operator $\circ$ satisfies the following associativity property:

$$
a \circ (b \circ c) = (a \circ b) \circ c
$$

where $a$, $b$, and $c$ are any two operators.

Lab Cell (SymPy):

```python
import sympy as sp

# Define the prefix operator

    return a * b

# Test the associativity property
x = sp.symbols('x')
y = sp.symbols('y')
z = sp.symbols('z')

assert sp.simplify(circ(circ(x, y), z)) == sp.simplify((circ(x, y) * z))
```

## Stem
### Definition

In mathematics, a stem is a part of an expression that contains the operation being performed. It is typically denoted by parentheses or other grouping symbols. The stem is the part of the expression that separates the operands (the values being operated on) from the operator.

For example, in the expression $2 + 3$, the stem is the $+$ symbol.

### Worked Example

Consider the expression $(x^2 + 1)/(x - 1)$. To evaluate this expression, we need to simplify it first. We can do this by using the distributive property of division over addition:

$$\frac{x^2 + 1}{x - 1} = \frac{(x^2)}{x - 1} + \frac{1}{x - 1}.$$

Now, we can evaluate each term separately. The first term is $\frac{x^2}{x-1}$.

### Key Theorem

If $f(x)$ and $g(x)$ are two functions, then the quotient of their compositions is equal to the composition of their quotients:

$$(f \circ g)(x) = (f/g)(x).$$

This theorem can be used to simplify complex expressions by rewriting them as a product of simpler expressions.

## In-

In calculus, a fundamental concept is the notion of *in*-functions, which generalizes the idea of inside a function. An in-function is defined as a function that satisfies certain properties related to its input values.

### Definition

Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be a function. We say that $f$ is an in-function if for every $x \in \mathbb{R}$, and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x_n) \rightarrow f(x). \]
This definition implies that the value of the function is constant when approaching a point from any direction.

### Worked Example

Consider the function $f: \mathbb{R} \rightarrow \mathbb{R}$ defined by $f(x) = |x|$. Let $\{x_n\}_{n=1}^\infty$ be a sequence of distinct real numbers converging to $0$. Then, we have
\[ f(x_n) = |x_n| \rightarrow 0. \]
However, if we consider the function $g: \mathbb{R} \rightarrow \mathbb{R}$ defined by $g(x) = -x$, then for any sequence $\{x_n\}_{n=1}^\infty$ converging to $-2$, we have
\[ g(x_n) = -(x_n) \rightarrow 2. \]
This example illustrates that the condition holds only when $f(x)$ is non-negative.

### Key Theorem

If $f: [a, b] \rightarrow [c, d]$ is an in-function, then for every $x \in (a, b)$ and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have
\[ f(x) = f(\lim_{n\to\infty} x_n). \]

```python
    return -x

```

## Port

A port is a mathematical object used to describe the boundary of a domain, particularly in the context of calculus and analysis. In essence, it represents the set of points that are "attached" or "connected" to the domain.

Formally, given a domain \(D\) in \(\mathbb{R}^n\), the port of \(D\) is denoted by \((\partial D)_P\) and defined as:
\[ (\partial D)_P = \left\{ x \in \partial D \mid \exists y \in D: d(x,y) > 0 \right\} \]
Here, \(d(x,y)\) denotes the Euclidean distance between points \(x\) and \(y\) in \(\mathbb{R}^n\), and \(\partial D\) is the boundary of the domain \(D\).

Worked Example:
Consider the unit square \(S = [0,1] \times [0,1]\) in \(\mathbb{R}^2\). The port of \(S\) is given by
\[ (\partial S)_P = \left\{ (x,y) \in \partial S: y > 0 \right\}. \]
To obtain this result, we can consider the unit square as consisting of two rectangles: one with vertices at \((0,0)\), \((1,0)\), and \((1,1)\), and another with vertices at \((0,0)\), \((0,1)\), and \((1,1)\). The port of \(S\) is then the set of points on the top edge of this square.

Key Theorem:
If \(D\) is a domain in \(\mathbb{R}^n\), then the port of \(D\) satisfies the following properties:

*   $(\partial D)_P$ is non-empty.
*   $$(\partial D)_P$$ is closed and bounded.
*   If $D$ is convex, then $(\partial D)_P$ is convex as well.

## Import

The `import` statement is a fundamental element in programming languages, used to bring external modules or packages into the current scope, making their definitions and functions available for use.

### Definition

In Python, for instance, the `import` statement allows programmers to utilize libraries or modules that contain collections of related functions, variables, and classes. The syntax for importing a module is as follows:

```latex
import module_name
```

For example, to import the `math` module in Python, one would use the following code:

```python
import math
```

Once imported, the contents of the module become accessible within the current scope.

### Worked Example

Consider a scenario where we want to perform mathematical calculations involving pi. We can utilize the `math` module from Python's standard library to achieve this:
```latex
import math

# Calculate pi using the math module
pi = math.pi

print(pi)  # Output: 3.14159265359...
```

In this example, we import the `math` module and then use its `pi` constant to calculate and print the value of pi.

### Key Theorem

The key theorem behind importing modules is that it enables programmers to reuse code written by others, reducing the need for redundant work and promoting modularity in their programs.
