### distance (0.1s)

## Distance

**Definition.** Let $\mathbf{u} = (u_1, u_2, \ldots, u_n)$ and $\mathbf{v} = (v_1, v_2, \ldots, v_n)$ be two points in $\mathbb{R}^n$. The **(Euclidean) distance** between $\mathbf{u}$ and $\mathbf{v}$ is defined by

$$
d(\mathbf{u}, \mathbf{v}) = \|\mathbf{u} - \mathbf{v}\| = \sqrt{\sum_{i=1}^{n}(u_i - v_i)^2}.
$$

This quantity generalises the familiar ruler measurement on a number line to any finite dimension and arises naturally from the inner product: $d(\mathbf{u}, \mathbf{v}) = \sqrt{\langle \mathbf{u}-\mathbf{v},\, \mathbf{u}-\mathbf{v}\rangle}$. Non-negativity ($d(\mathbf{u},\mathbf{v}) \geq 0$, with equality if and only if $\mathbf{u} = \mathbf{v}$) and symmetry ($d(\mathbf{u},\mathbf{v}) = d(\mathbf{v},\mathbf{u})$) follow immediately from the definition; both are left as short exercises.

**Worked Example.** Compute the distance between $\mathbf{u} = (1, 3, -2)$ and $\mathbf{v} = (4, -1, 2)$ in $\mathbb{R}^3$.

$$
\mathbf{u} - \mathbf{v} = (1-4,\; 3-(-1),\; -2-2) = (-3,\; 4,\; -4).
$$

$$
d(\mathbf{u}, \mathbf{v}) = \sqrt{(-3)^2 + 4^2 + (-4)^2} = \sqrt{9 + 16 + 16} = \sqrt{41} \approx 6.403.
$$

**Key Theorem (Triangle Inequality).** For all $\mathbf{u}, \mathbf{v}, \mathbf{w} \in \mathbb{R}^n$,

$$
d(\mathbf{u}, \mathbf{w}) \leq d(\mathbf{u}, \mathbf{v}) + d(\mathbf{v}, \mathbf{w}).
$$

Geometrically, no side of a triangle is longer than the sum of the other two. This is the non-trivial metric axiom and the one with genuine content to prove.

*Proof sketch.* By the Cauchy–Schwarz inequality, $|\langle \mathbf{x}, \mathbf{y}\rangle| \leq \|\mathbf{x}\|\,\|\mathbf{y}\|$. Set $\mathbf{x} = \mathbf{u}-\mathbf{v}$ and $\mathbf{y} = \mathbf{v}-\mathbf{w}$. Expanding $\|\mathbf{u}-\mathbf{w}\|^2 = \|(\mathbf{u}-\mathbf{v})+(\mathbf{v}-\mathbf{w})\|^2$ and applying the inequality to the cross term yields $\|\mathbf{u}-\mathbf{w}\|^2 \leq \bigl(\|\mathbf{u}-\mathbf{v}\|+\|\mathbf{v}-\mathbf{w}\|\bigr)^2$; taking square roots gives the result. $\blacksquare$

**Lab Cell (SymPy).**

```python
from sympy import Matrix, sqrt, simplify

u = Matrix([1, 3, -2])
v = Matrix([4, -1, 2])

diff = u - v
dist = sqrt(diff.dot(diff))
print("Distance:", simplify(dist))   # sqrt(41)
```

Running this cell confirms $d(\mathbf{u}, \mathbf{v}) = \sqrt{41}$. Experiment by replacing the coordinates to build intuition for how each component contributes quadratically to the total separation.

### point (0.1s)

## Point

**Definition.** A *point* is the most primitive object in Euclidean geometry: it has no dimension, no length, no area, and no volume. Formally, in $\mathbb{R}^n$, a point is an ordered $n$-tuple

$$P = (x_1, x_2, \ldots, x_n) \in \mathbb{R}^n,$$

where each $x_i \in \mathbb{R}$ is called a *coordinate* of $P$ along the $i$-th axis. In the plane ($n = 2$) we write $P = (x, y)$; in space ($n = 3$) we write $P = (x, y, z)$. A point is location without extent — it satisfies Euclid's first postulate: *"A point is that which has no part."*

**Worked Example.** Let $P = (3, -1)$ and $Q = (7, 2)$ be two points in $\mathbb{R}^2$. The *distance* between them is given by the Euclidean metric

$$d(P, Q) = \sqrt{(7 - 3)^2 + (2 - (-1))^2} = \sqrt{16 + 9} = \sqrt{25} = 5.$$

Observe that $d(P, Q) = d(Q, P)$, and that $d(P, Q) = 0$ if and only if $P$ and $Q$ are the same point — properties that make the Euclidean metric a proper *metric* in the sense of analysis. The distance of 5 units here is exact; no rounding occurs because the radicand is a perfect square.

**Key Theorem (Distance Formula).** For any two points $P = (x_1, y_1)$ and $Q = (x_2, y_2)$ in $\mathbb{R}^2$,

$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.$$

*Proof sketch.* Drop a perpendicular from $Q$ to the horizontal line through $P$, forming a right triangle with legs $|x_2 - x_1|$ and $|y_2 - y_1|$. The hypotenuse equals $d(P, Q)$; the result follows directly from the Pythagorean theorem. The formula extends to $\mathbb{R}^n$ by induction, adding one squared-difference term per additional coordinate. $\blacksquare$

**Lab Cell (SymPy).**

```python
from sympy import sqrt

x1, y1, x2, y2 = 3, -1, 7, 2
distance = sqrt((x2 - x1)**2 + (y2 - y1)**2)

print("Distance :", distance)   # 5
```

Running this cell confirms the computation symbolically, illustrating how SymPy handles exact arithmetic — returning the integer $5$ rather than the floating-point approximation $5.0$. This distinction matters whenever subsequent steps require algebraic manipulation of the result.

### circle (82.7s)

## Circle

**Definition.** A *circle* is the set of all points in the Euclidean plane equidistant from a fixed point called the *centre*. Formally, given a centre $C = (h, k) \in \mathbb{R}^2$ and a radius $r > 0$, the circle is

$$\mathcal{C} = \{\, (x, y) \in \mathbb{R}^2 \mid (x - h)^2 + (y - k)^2 = r^2 \,\}.$$

This equation is the **standard form**. Expanding and collecting constants yields the **general form**

$$x^2 + y^2 + Dx + Ey + F = 0,$$

where $D = -2h$, $E = -2k$, and $F = h^2 + k^2 - r^2$. Conversely, any equation of this form with $D^2 + E^2 - 4F > 0$ defines a circle; completing the square recovers $h$, $k$, and $r$.

---

**Worked Example.** Identify the centre and radius of $x^2 + y^2 - 6x + 4y - 3 = 0$.

Group and complete the square in each variable:

$$\underbrace{(x^2 - 6x + 9)}_{(x-3)^2} + \underbrace{(y^2 + 4y + 4)}_{(y+2)^2} = 3 + 9 + 4 = 16.$$

Hence $(x-3)^2 + (y+2)^2 = 4^2$, giving centre $(3, -2)$ and radius $r = 4$.

---

**Key Theorem (Tangent–Radius Perpendicularity).** A *tangent line* to $\mathcal{C}$ is a line that meets the circle at exactly one point. If $T$ is that point and $O$ is the centre, then the radius $OT$ is perpendicular to the tangent.

*Proof sketch.* Suppose $OT$ were not perpendicular to the tangent line $\ell$. Let $F$ be the foot of the perpendicular from $O$ to $\ell$. Then $|OF| < |OT| = r$, so $F$ lies strictly inside $\mathcal{C}$. Because $\ell$ passes through $F$ and extends in both directions, the Intermediate Value Theorem guarantees a second intersection of $\ell$ with $\mathcal{C}$, contradicting the definition of tangent. $\square$

---

**Lab Cell (SymPy).**

```python
from sympy import symbols, expand, sqrt

x, y = symbols('x y')
eq = x**2 + y**2 - 6*x + 4*y - 3          # general form
# Complete the square to extract centre and radius
h, k = 3, -2
r_sq = 16
print(f"Centre: ({h}, {k}),  Radius: {sqrt(r_sq)}")

# Verify: point (7, -2) lies on the circle
print(expand((7 - h)**2 + (-2 - k)**2))    # expected: 16

# Tangent at (7, -2) is the vertical line x = 7;
# distance from centre to that line must equal r.
print(abs(7 - h))                          # expected: 4
```

Running this cell prints `Centre: (3, -2), Radius: 4`, confirms that $(7, -2)$ lies on $\mathcal{C}$, and verifies that the distance from the centre to the tangent line at that point equals the radius — a direct numerical check of the theorem above.
