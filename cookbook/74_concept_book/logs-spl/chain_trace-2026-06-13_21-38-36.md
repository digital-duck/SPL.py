### line (0.1s)

## Line

**Definition.** A *line* in the Euclidean plane $\mathbb{R}^2$ is a one-dimensional affine subspace. Formally, given a point $P_0 = (x_0, y_0)$ and a nonzero direction vector $\mathbf{d} = (d_1, d_2)$, the line $\ell$ is the set

$$\ell = \bigl\{(x_0 + t\,d_1,\; y_0 + t\,d_2) : t \in \mathbb{R}\bigr\}.$$

This is the **parametric form**. Eliminating $t$ by solving the first component for $t$ and substituting into the second yields the equivalent **implicit (Cartesian) form**

$$ax + by = c, \qquad (a,b) \neq (0,0),$$

where $a = d_2$, $b = -d_1$, and $c = d_2 x_0 - d_1 y_0$. The implicit form is often more convenient for checking whether a given point lies on the line: substitute the coordinates and verify the equality.

---

**Worked Example.** Find the equation of the line passing through $A = (1, 3)$ and $B = (4, 9)$.

*Direction vector:* $\mathbf{d} = B - A = (3, 6)$, so we may take $(d_1, d_2) = (1, 2)$.

*Parametric form* anchored at $A$:

$$(x, y) = (1 + t,\; 3 + 2t), \quad t \in \mathbb{R}.$$

*Implicit form:* $a = 2$, $b = -1$, $c = 2(1) - 1(3) = -1$, giving $2x - y = -1$, or equivalently $y = 2x + 1$.

Verification: $y(4) = 2(4) + 1 = 9$ ✓.

---

**Key Theorem (Unique Line Through Two Points).** *For any two distinct points $P, Q \in \mathbb{R}^2$, there exists exactly one line containing both.*

*Proof sketch.* Existence: take $\mathbf{d} = Q - P$ and use the parametric form above. Uniqueness: if two lines $\ell_1, \ell_2$ both contain $P$ and $Q$, their direction vectors are both parallel to $Q - P$, hence they share a point and a direction, so $\ell_1 = \ell_2$. $\square$

---

**Lab Cell (SymPy)**

```python
from sympy import symbols, solve, Rational

x, y = symbols('x y')

# Two points
x0, y0 = 1, 3
x1, y1 = 4, 9

# Direction vector components
d1, d2 = x1 - x0, y1 - y0          # (3, 6), simplified to (1, 2)

# Implicit form: a*x + b*y = c
a, b = d2, -d1
c = d2 * x0 - d1 * y0

implicit_eq = a*x + b*y - c
print(f"{a}*x + ({b})*y = {c}")     # 6*x + (-3)*y = -3, i.e. 2x - y = -1

# Solve for x given y = 7
sol = solve(implicit_eq.subs(y, 7), x)
print(f"x when y=7: {sol[0]}")      # x = 3
```

### point (0.1s)

## Point

**Definition.** A *point* is the most primitive object in Euclidean geometry: it has no dimension, no length, no area, and no volume. Formally, in $\mathbb{R}^n$, a point is an ordered $n$-tuple

$$P = (x_1, x_2, \ldots, x_n) \in \mathbb{R}^n,$$

where each $x_i \in \mathbb{R}$ is called a *coordinate* of $P$ along the $i$-th axis. In the plane ($n = 2$) we write $P = (x, y)$; in space ($n = 3$) we write $P = (x, y, z)$. A point is location without extent — it satisfies Euclid's first postulate: *"A point is that which has no part."*

**Worked Example.** Let $P = (3, -1)$ and $Q = (7, 2)$ be two points in $\mathbb{R}^2$. The *distance* between them is given by the Euclidean metric

$$d(P, Q) = \sqrt{(7 - 3)^2 + (2 - (-1))^2} = \sqrt{16 + 9} = \sqrt{25} = 5.$$

Observe that $d(P, Q) = d(Q, P)$, and that $d(P, Q) = 0$ if and only if $P$ and $Q$ are the same point — properties that make the Euclidean metric a proper *metric* in the sense of analysis. The distance of 5 units here is exact; no rounding occurs because the radicand is a perfect square.

**Key Theorem (Distance Formula).** For any two points $P = (x_1, y_1)$ and $Q = (x_2, y_2)$ in $\mathbb{R}^2$,

$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.$$

*Proof sketch.* Drop a perpendicular from $Q$ to the horizontal line through $P$, forming a right triangle with legs $|x_2 - x_1|$ and $|y_2 - y_1|$. The hypotenuse equals $d(P, Q)$; the result follows directly from the Pythagorean theorem. The formula extends to $\mathbb{R}^n$ by induction, adding one squared-difference term per additional coordinate. $\blacksquare$

**Lab Cell (SymPy).**

```python
from sympy import sqrt

x1, y1, x2, y2 = 3, -1, 7, 2
distance = sqrt((x2 - x1)**2 + (y2 - y1)**2)

print("Distance :", distance)   # 5
```

Running this cell confirms the computation symbolically, illustrating how SymPy handles exact arithmetic — returning the integer $5$ rather than the floating-point approximation $5.0$. This distinction matters whenever subsequent steps require algebraic manipulation of the result.

### ray (0.1s)

## Ray

**Definition.** A *ray* is a subset of a line that begins at a fixed point and extends infinitely in one direction. Formally, given two distinct points $A$ and $B$ in $\mathbb{R}^n$, the ray with *endpoint* (or *origin*) $A$ passing through $B$ is defined as

$$\overrightarrow{AB} = \{ A + t\,(B - A) \mid t \in [0, +\infty) \}.$$

The point $A$ belongs to the ray; no point "behind" $A$ (i.e., corresponding to $t < 0$) does. The ray is thus a half-line closed at $A$. In the plane $\mathbb{R}^2$, a ray is often specified by its endpoint and a direction vector $\mathbf{d} \neq \mathbf{0}$:

$$r(t) = P + t\,\mathbf{d}, \qquad t \geq 0.$$

**Worked Example.** Let $P = (1, 2)$ and $\mathbf{d} = (3, -1)$ in $\mathbb{R}^2$. Find the point on the ray at parameter $t = 2$, and determine whether $Q = (7, 0)$ lies on the ray.

*Point at $t = 2$:*
$$r(2) = (1,2) + 2\,(3,-1) = (1 + 6,\; 2 - 2) = (7, 0).$$

*Membership check for $Q = (7,0)$:* Solve $P + t\,\mathbf{d} = Q$:
$$t = \frac{7 - 1}{3} = 2, \qquad t = \frac{0 - 2}{-1} = 2.$$
Both components yield $t = 2 \geq 0$, so $Q \in \overrightarrow{AB}$. $\square$

**Key Theorem (Ray–Ray Intersection).** Two rays $r_1(t) = P + t\,\mathbf{d}_1$ and $r_2(s) = Q + s\,\mathbf{d}_2$ in $\mathbb{R}^2$ intersect if and only if there exist $t \geq 0$ and $s \geq 0$ satisfying
$$P + t\,\mathbf{d}_1 = Q + s\,\mathbf{d}_2.$$
This $2 \times 2$ linear system has a unique solution when $\mathbf{d}_1$ and $\mathbf{d}_2$ are not parallel (i.e., $\mathbf{d}_1 \times \mathbf{d}_2 \neq 0$, the scalar 2D cross product), and the intersection is valid precisely when both parameters are non-negative.

**Lab Cell (SymPy).**
```python
from sympy import symbols, Matrix, solve

t, s = symbols('t s', real=True)
P = Matrix([0, 0]);  d1 = Matrix([1, 1])
Q = Matrix([2, 0]);  d2 = Matrix([-1, 1])

eq = P + t*d1 - (Q + s*d2)
sol = solve([eq[0], eq[1]], [t, s])
print(sol)          # {t: 1, s: 1}  → intersection at (1, 1)
assert sol[t] >= 0 and sol[s] >= 0
```

Running this cell confirms $t = 1 \geq 0$ and $s = 1 \geq 0$, so the two rays meet at $r_1(1) = (1, 1)$.
