**Question:** A ball is thrown upward with an initial velocity, and its height can be modeled by a quadratic equation. Solve for x in the equation x² + x - 2 = 0.

**Solution:** 1.  We have the quadratic equation x² + x - 2 = 0. To solve for *x*, we can factor this equation.

2.  We need to find two numbers that multiply to -2 and add up to 1. These numbers are 2 and -1. Therefore, we can rewrite the middle term as:
    x² + 2x - x - 2 = 0

3.  Now, factor by grouping:
    x(x + 2) - 1(x + 2) = 0
    (x - 1)(x + 2) = 0

4.  Set each factor equal to zero and solve for *x*:
     x - 1 = 0   or   x + 2 = 0
     x = 1       or   x = -2

exact_answer: x = -2 or x = 1

**Question:** A ball is thrown upward with an initial velocity and then falls back down to Earth.  Solve for x: x² - 5x + 6 = 0

**Solution:** 1.  We have the quadratic equation x² - 5x + 6 = 0. To solve for *x*, we can factor the quadratic expression.

2.  We are looking for two numbers that multiply to 6 and add up to -5. These numbers are -2 and -3. Therefore, we can rewrite the equation as (x - 2)(x - 3) = 0.

3.  For the product of two factors to be zero, at least one of the factors must be zero. So, either x - 2 = 0 or x - 3 = 0.

4.  Solving these equations gives us x = 2 and x = 3.
