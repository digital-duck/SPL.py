### distance (18.7s)

## Distance

在日常生活中，我们经常需要回答一个基本问题：两个地点之间有多远？无论是导航地图上的路线规划，还是工程师计算桥梁的跨度，"距离"这个概念都是空间推理的基石。

**定义。** 在 $\mathbb{R}^n$ 空间中，两点 $P = (p_1, p_2, \dots, p_n)$ 与 $Q = (q_1, q_2, \dots, q_n)$ 之间的**欧几里得距离**定义为：

$$d(P, Q) = \sqrt{\sum_{i=1}^{n}(q_i - p_i)^2}$$

距离函数（度量）满足三条公理：非负性 $d(P,Q) \geq 0$（等号成立当且仅当 $P=Q$）；对称性 $d(P,Q) = d(Q,P)$；以及三角不等式 $d(P,R) \leq d(P,Q) + d(Q,R)$。

**计算示例。** 设平面上 $A = (1, 2)$，$B = (4, 6)$。则：

$$d(A, B) = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

这正是一个以 $3$ 和 $4$ 为直角边的直角三角形的斜边长度。

**为什么成立？** 距离公式本质上是勾股定理的直接推广。在二维情形中，从 $A$ 到 $B$ 的水平位移为 $\Delta x = q_1 - p_1$，垂直位移为 $\Delta y = q_2 - p_2$，它们构成直角三角形的两条直角边。由勾股定理 $c^2 = a^2 + b^2$，斜边 $c = \sqrt{(\Delta x)^2 + (\Delta y)^2}$ 即为两点间的直线距离。推广到 $n$ 维时，沿每个坐标轴的位移彼此正交，因此可以逐维叠加平方项，原理完全一致。

**练习。** 在三维空间中，计算 $P = (2, -1, 3)$ 与 $Q = (5, 3, -1)$ 之间的距离。验证你的结果满足 $d(P,Q) = d(Q,P)$，并思考：如果存在第三个点 $R = (2, -1, -1)$，三角不等式 $d(P,Q) \leq d(P,R) + d(R,Q)$ 是否成立？

### point (18.2s)

## Point

你有没有想过，地图上标注的一座城市，其实只是一个小圆点？无论你把地图放大多少倍，那个标记代表的是一个**位置**，而不是一块区域。这就是数学中"点"的核心思想——它只有位置，没有大小。

**定义。** 在欧几里得几何中，**点**（point）是最基本的未定义元素。点没有长度、宽度或厚度——它是零维的。我们用一个大写字母来命名一个点，例如 $A$、$B$、$P$。在坐标几何中，平面上的一个点由一个有序对 $(x, y)$ 唯一确定，其中 $x$ 是横坐标，$y$ 是纵坐标。

**示例。** 考虑点 $P = (3, 4)$。它告诉我们：从原点 $O = (0, 0)$ 出发，沿 $x$ 轴正方向移动 $3$ 个单位，再沿 $y$ 轴正方向移动 $4$ 个单位，就到达 $P$。我们可以计算 $P$ 到原点的距离：

$$d(O, P) = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

注意，这个距离是一个数值，而 $P$ 本身仍然是一个没有大小的位置。

**为什么点是"无定义"的？** 欧几里得在《几何原本》中写道："点是没有部分的东西。"这并非偷懒，而是深刻的逻辑选择。几何体系需要起点——如果每个概念都要用更基本的概念来定义，就会陷入无穷倒退。点、直线和平面作为**原始概念**（primitive notions），它们的性质不通过定义获得，而是通过**公理**来约束。例如，公理"过两点恰好有一条直线"就赋予了点与直线之间的关系，而无需先"定义"它们各自是什么。

**练习。** 给定点 $A = (1, 2)$ 和 $B = (4, 6)$，求线段 $AB$ 的中点 $M$ 的坐标，并验证 $d(A, M) = d(M, B)$。

### circle (17.9s)

## Circle

你有没有注意过，往平静的湖面扔一颗石子，水波会以落点为中心向四周均匀扩散？每一圈波纹上的水点到落点的距离都相同——这正是圆的本质。

**定义：** 在平面上，到定点 $O$（圆心）的距离等于定长 $r$（半径，$r > 0$）的所有点的集合，称为**圆**。用集合语言表达：

$$\mathcal{C} = \{P \in \mathbb{R}^2 \mid |PO| = r\}$$

若圆心在坐标原点，圆的方程为 $x^2 + y^2 = r^2$；若圆心在 $(a, b)$，则为

$$(x - a)^2 + (y - b)^2 = r^2$$

**例题：** 判断点 $P(3, 4)$ 是否在圆心为 $O(0, 0)$、半径为 $5$ 的圆上。

计算 $P$ 到 $O$ 的距离：

$$|PO| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

因为 $|PO| = r = 5$，所以 $P$ 在圆上。$\square$

这个例子也揭示了圆与勾股定理的深层联系：$3^2 + 4^2 = 5^2$ 既是一组勾股数，也恰好满足圆的方程。

**为什么圆的方程是这个形式？** 圆的定义要求"到圆心距离恒为 $r$"。在坐标系中，两点间距离由勾股定理给出：$\sqrt{(x-a)^2 + (y-b)^2} = r$。两边平方，即得标准方程。方程的每一步都直接来自距离的定义——没有额外假设，只有逻辑推导。这说明圆的代数表达与几何定义是完全等价的。

圆还有一个重要的度量性质：周长 $C = 2\pi r$，面积 $A = \pi r^2$。常数 $\pi \approx 3.14159$ 是周长与直径之比，它的存在保证了所有圆在几何上是**相似**的——大小不同，形状完全一致。

**练习：** 圆心为 $(2, -1)$，且过点 $(5, 3)$ 的圆，其半径是多少？写出该圆的标准方程。
