### distance (19.1s)

## Distance

### 现实中的距离

当你用导航软件查找从家到学校的路线时，屏幕上显示的公里数就是一种"距离"。但数学中的距离远不止于此——它是度量空间中最基本的概念，为我们提供了衡量"两个对象有多远"的严格框架。

### 定义

给定集合 $X$，一个函数 $d: X \times X \to \mathbb{R}$ 称为 **距离函数**（或度量），如果对所有 $x, y, z \in X$，满足：

1. **非负性**：$d(x, y) \geq 0$，且 $d(x, y) = 0 \iff x = y$
2. **对称性**：$d(x, y) = d(y, x)$
3. **三角不等式**：$d(x, z) \leq d(x, y) + d(y, z)$

最常见的距离是欧几里得距离。在 $\mathbb{R}^n$ 中，两点 $\mathbf{p} = (p_1, \dots, p_n)$ 与 $\mathbf{q} = (q_1, \dots, q_n)$ 之间的距离为：

$$d(\mathbf{p}, \mathbf{q}) = \sqrt{\sum_{i=1}^{n}(p_i - q_i)^2}$$

### 计算示例

设 $\mathbf{p} = (1, 2)$，$\mathbf{q} = (4, 6)$，求 $d(\mathbf{p}, \mathbf{q})$。

$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

验证三角不等式：取 $\mathbf{r} = (4, 2)$，则 $d(\mathbf{p}, \mathbf{r}) = 3$，$d(\mathbf{r}, \mathbf{q}) = 4$。果然 $5 \leq 3 + 4 = 7$，不等式成立。

### 为什么三角不等式成立？

直觉上，两点之间直线最短——绕道经过第三点只会更远。严格证明依赖柯西-施瓦茨不等式：对向量 $\mathbf{u}, \mathbf{v}$，有 $|\mathbf{u} \cdot \mathbf{v}| \leq \|\mathbf{u}\| \|\mathbf{v}\|$，由此可推导出 $\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$，这正是三角不等式的向量形式。

### 练习

在 $\mathbb{R}^3$ 中，设 $A = (0, 0, 0)$，$B = (1, 2, 2)$，$C = (3, 6, 0)$。

1. 计算 $d(A, B)$ 和 $d(A, C)$。
2. 验证 $d(A, C) \leq d(A, B) + d(B, C)$ 是否成立。

### point (19.8s)

## Point

在日常生活中，我们用笔尖在纸上轻轻一触，留下一个墨迹——这就是"点"最直观的原型。地图上的城市、棋盘上的落子、屏幕上的像素，都是点的化身。然而，数学中的点比这些都更为纯粹：它没有大小、没有形状，只有**位置**。

**定义。** 在欧几里得几何中，**点**（point）是最基本的未定义术语（undefined term）。它表示空间中一个确定的位置，没有长度、宽度或高度。我们通常用大写字母标记一个点，如 $A$、$B$、$P$。在坐标系中，一个点由一组有序数对（二维）或有序数组（三维）唯一确定：

$$P = (x, y) \quad \text{（二维）}, \qquad P = (x, y, z) \quad \text{（三维）}$$

**实例。** 在平面直角坐标系中，点 $A = (3, 4)$ 表示从原点 $O = (0,0)$ 向右移动 $3$ 个单位、再向上移动 $4$ 个单位所到达的位置。点 $A$ 到原点的距离为：

$$d(O, A) = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = 5$$

这个计算本身就揭示了点的核心角色——它是度量距离、定义图形的起点。

**为什么点是基础？** 欧几里得在《几何原本》中将点定义为"没有部分的东西"（*that which has no part*）。这并非随意之举：几何中所有更复杂的对象——线、面、体——都由点构建而来。两点确定一条直线，三个不共线的点确定一个平面。没有点，就没有位置；没有位置，就无法谈论形状与空间。点是几何大厦的原子。

**练习。** 给定平面上两点 $P = (1, 2)$ 和 $Q = (4, 6)$，求线段 $PQ$ 的中点 $M$ 的坐标，并验证 $d(P, M) = d(M, Q)$。

### circle (16.5s)

## Circle

你有没有注意过，往平静的水面扔一颗石子，涟漪总是呈完美的圆形向外扩散？这是因为波在各个方向上以相同的速度传播——每一点到中心的距离都相等。这个简单的观察，正是圆的数学本质。

**定义：** 平面上到定点 $O$（圆心）距离等于定长 $r$（半径）的所有点的集合，称为**圆**。用坐标表示，以 $(a, b)$ 为圆心、$r$ 为半径的圆的标准方程为：

$$(x - a)^2 + (y - b)^2 = r^2$$

这个方程直接来源于两点间的距离公式。点 $(x, y)$ 在圆上，当且仅当它到圆心的距离恰好为 $r$。

**例题：** 求以 $(3, -2)$ 为圆心、半径为 $5$ 的圆的方程，并判断点 $P(6, 2)$ 是否在圆上。

标准方程为：

$$(x - 3)^2 + (y + 2)^2 = 25$$

将 $P(6, 2)$ 代入左边：

$$(6 - 3)^2 + (2 + 2)^2 = 9 + 16 = 25$$

结果等于 $r^2 = 25$，因此 $P$ 恰好在圆上。若计算结果小于 $25$，则点在圆内；大于 $25$，则在圆外。

**为什么成立？** 圆的方程本质上是勾股定理的直接应用。点 $(x, y)$ 与圆心 $(a, b)$ 之间的水平距离为 $|x - a|$，垂直距离为 $|y - b|$，它们构成直角三角形的两条直角边。斜边就是点到圆心的距离，由勾股定理得 $\sqrt{(x-a)^2 + (y-b)^2} = r$，两边平方即得标准方程。圆的完美对称性——旋转不变性——正源于距离的定义对方向没有偏好。

**练习：** 已知圆的方程为 $x^2 + y^2 - 6x + 4y + 4 = 0$，通过配方将其化为标准形式，求出圆心坐标和半径。
