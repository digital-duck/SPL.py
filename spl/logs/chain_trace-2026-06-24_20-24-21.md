## Distance

在日常生活中，我们时刻在估算距离：从家到学校有多远？两座城市之间隔了多少公里？这种"两点之间有多远"的直觉，正是数学中**距离**概念的起点。

**定义。** 在 $\mathbb{R}^n$ 中，两点 $\mathbf{p} = (p_1, p_2, \dots, p_n)$ 与 $\mathbf{q} = (q_1, q_2, \dots, q_n)$ 之间的**欧几里得距离**（Euclidean distance）定义为

$$d(\mathbf{p}, \mathbf{q}) = \sqrt{\sum_{i=1}^{n} (p_i - q_i)^2}$$

一个合格的距离函数 $d$ 必须满足三条公理：
1. **非负性与同一性：** $d(\mathbf{p}, \mathbf{q}) \geq 0$，且 $d(\mathbf{p}, \mathbf{q}) = 0$ 当且仅当 $\mathbf{p} = \mathbf{q}$。
2. **对称性：** $d(\mathbf{p}, \mathbf{q}) = d(\mathbf{q}, \mathbf{p})$。
3. **三角不等式：** $d(\mathbf{p}, \mathbf{r}) \leq d(\mathbf{p}, \mathbf{q}) + d(\mathbf{q}, \mathbf{r})$。

**示例。** 求平面上 $A = (1, 2)$ 与 $B = (4, 6)$ 之间的距离。

$$d(A, B) = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**为什么成立？** 欧几里得距离本质上是勾股定理的推广。在二维情形下，两点的水平差 $\Delta x = 3$ 和垂直差 $\Delta y = 4$ 构成一个直角三角形的两条直角边，斜边长度恰好就是 $\sqrt{\Delta x^2 + \Delta y^2}$。推广到 $n$ 维时，我们沿每个坐标轴分别计算差值，再用同样的"求和取根号"操作合成总距离——这正是高维空间中勾股定理的自然延伸。三角不等式的成立则源于柯西-施瓦茨不等式，它保证了"绕路永远不会更短"。

**练习。** 在三维空间中，求 $P = (1, 0, -2)$ 与 $Q = (3, 4, 1)$ 之间的距离，并验证 $d(P, Q) = d(Q, P)$。
