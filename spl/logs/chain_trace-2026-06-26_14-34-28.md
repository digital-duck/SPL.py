## Scalar Multiplication

**Definition:** Given a vector space $V$ over a field $\mathbb{F}$ (typically $\mathbb{R}$ or $\mathbb{C}$), scalar multiplication is an operation that takes a scalar $\alpha \in \mathbb{F}$ and a vector $\mathbf{v} \in V$, and produces a new vector $\alpha\mathbf{v} \in V$. This operation must satisfy two core properties to preserve the structure of $V$ as a vector space:
1.  Compatibility with field multiplication: $(\alpha\beta)\mathbf{v} = \alpha(\beta\mathbf{v})$ for all $\alpha, \beta \in \mathbb{F}$.
2.  Distributivity over scalar addition: $(\alpha + \beta)\mathbf{v} = \alpha\mathbf{v} + \beta\mathbf{v}$ for all $\alpha, \beta \in \mathbb{F}$.

For a vector $\mathbf{v}$ in $\mathbb{R}^n$, the operation is defined component-wise. If $\mathbf{v} = (v_1, v_2, \ldots, v_n)$ and $\alpha$ is a scalar, then the resulting vector is:
$$\alpha\mathbf{v} = (\alpha v_1, \alpha v_2, \ldots, \alpha v_n)$$

**Proof Sketch (In $\mathbb{R}^2$):** We must show that the operation $\alpha(x, y) = (\alpha x, \alpha y)$ is well-defined and preserves vector space axioms. Let $\mathbf{u} = (x_1, y_1)$ and $\mathbf{v} = (x_2, y_2)$, and let $\alpha, \beta$ be scalars.
First, we verify the compatibility of scalar multiplication with vector addition: $\alpha(\mathbf{u} + \mathbf{v}) = \alpha(x_1+x_2, y_1+y_2) = (\alpha(x_1+x_2), \alpha(y_1+y_2))$. By the distributive property in $\mathbb{R}$, this equals $(\alpha x_1 + \alpha x_2, \alpha y_1 + \alpha y_2)$, which is precisely $(\alpha \mathbf{u} + \alpha \mathbf{v})$. This confirms that scalar multiplication interacts correctly with vector addition.

**Worked Example:** Consider the vector $\mathbf{v} = (3, -1)$ in $\mathbb{R}^2$ and the scalar $\alpha = 4$. The scalar multiple is calculated as:
$$4\mathbf{v} = 4(3, -1) = (4 \cdot 3, 4 \cdot (-1)) = (12, -4)$$
If we use a negative scalar, say $\beta = -2$, the operation yields:
$$-2\mathbf{v} = -2(3, -1) = (-6, 2)$$
This demonstrates that multiplying by a negative scalar changes the direction of the vector while preserving its magnitude.

## Vector Addition

Vector addition is the process of combining two or more vectors to find a single resultant vector that represents the net effect of all individual components. Given two vectors, $\vec{A}$ and $\vec{B}$, their sum, denoted $\vec{R}$, must satisfy the principle of superposition. Geometrically, this can be visualized using the tip-to-tail rule (or Head-to-Tail method), where $\vec{A}$ is drawn first, and $\vec{B}$ is then drawn starting from the head of $\vec{A}$. The resultant vector $\vec{R}$ connects the tail of $\vec{A}$ to the head of $\vec{B}$. Algebraically, this process simplifies significantly by utilizing a Cartesian coordinate system. If a vector $\vec{V}$ has components $(V_x, V_y, V_z)$, the addition is performed component-wise:

$$\vec{R} = \vec{A} + \vec{B} = (A_x + B_x)\hat{\mathbf{i}} + (A_y + B_y)\hat{\mathbf{j}} + (A_z + B_z)\hat{\mathbf{k}}$$

This ensures that the magnitude and direction of $\vec{R}$ are accurately determined without needing to solve complex trigonometric relationships repeatedly.

## Linear Combination

A linear combination is the fundamental operation used to construct new vectors within a given vector space by scaling existing basis vectors. Formally, let $V$ be a vector space over a field of scalars $\mathbb{F}$ (typically $\mathbb{R}$). $$ w = c_1 v_1 + c_2 v_2 + \cdots + c_k v_k = \sum_{i=1}^{k} c_i v_i $$
where $c_1, c_2, \dots, c_k$ are scalars belonging to the field $\mathbb{F}$. Each scalar $c_i$ determines the magnitude and direction of the contribution of the corresponding vector $v_i$. ## Linear Map

A linear map, or linear transformation, is a function between two vector spaces that preserves the operations of vector addition and scalar multiplication. Given two vector spaces $V$ and $W$ over the same field $\mathbb{F}$ (typically $\mathbb{R}$), a mapping $T: V \to W$ is defined as linear if it satisfies two fundamental conditions for all vectors $\mathbf{u}, \mathbf{v} \in V$ and all scalars $c \in \mathbb{F}$:

1.  **Additivity:** $T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})$
2.  **Homogeneity (Scalar Property):** $T(c\mathbf{u}) = cT(\mathbf{u})$

These two properties can be combined into a single, equivalent condition: $T(c_1\mathbf{u} + c_2\mathbf{v}) = c_1T(\mathbf{u}) + c_2T(\mathbf{v})$. Crucially, every linear map must map the zero vector to the zero vector, as $T(\mathbf{0}_V) = T(0 \cdot \mathbf{v}) = 0 \cdot T(\mathbf{v}) = \mathbf{0}_W$.

**Worked Example:**
Consider $\mathbb{R}^3$ and $\mathbb{R}^2$, both vector spaces over $\mathbb{R}$. Define a function $T: \mathbb{R}^3 \to \mathbb{R}^2$ by $T(x, y, z) = (xy + z, x^2 - y)$.
To test for linearity, we must check the scalar property. Let $\mathbf{u} = (1, 0, 0)$ and $c=2$.
$T(c\mathbf{u}) = T((2, 0, 0)) = ((2)(0) + 0, (2)^2 - 0) = (0, 4)$.
However, $cT(\mathbf{u}) = 2 \cdot T((1, 0, 0)) = 2 \cdot (0, 1) = (0, 2)$.
Since $(0, 4) \neq (0, 2)$, the function $T$ is **not** a linear map. A correct example would be $L(x, y, z) = (ax + by + cz, dx + ey + fz)$, which always satisfies the linearity conditions.

**Key Theorem (Matrix Representation):**
A fundamental result in linear algebra states that if $V$ and $W$ are finite-dimensional vector spaces over $\mathbb{F}$, then every linear map $T: V \to W$ can be uniquely represented by a matrix multiplication. Specifically, if we choose bases for $V$ and $W$, there exists an $m \times n$ matrix $[A]$ such that $T(\mathbf{v}) = [A] \mathbf{v}$. This theorem allows us to translate the abstract concept of linearity into concrete matrix algebra.

## Eigenpair

An eigenpair constitutes a pair of values $(\lambda, v)$ associated with a square matrix $A$, where $v$ is a non-zero vector and $\lambda$ is a scalar. Formally, $\lambda$ is called an eigenvalue, and the corresponding non-zero vector $v$ is called the eigenvector. The relationship between these components is defined by the fundamental equation:
$$Av = \lambda v$$
This defining equation expresses that when the matrix $A$ acts upon the eigenvector $v$, the resulting vector $Av$ is simply a scalar multiple ($\lambda$) of the original vector $v$. Geometrically, this means that the action of the linear transformation represented by $A$ only stretches or shrinks the eigenvector $v$; it does not change its direction (assuming $\lambda \neq 0$). To find these pairs, we rearrange the equation:
$$Av - \lambda v = 0$$
$$(A - \lambda I)v = 0$$
For a non-trivial solution ($v \neq 0$) to exist, the matrix $(A - \lambda I)$ must be singular, meaning its determinant must vanish: $\det(A - \lambda I) = 0$. This equation is known as the characteristic equation, and its roots provide the eigenvalues $\lambda$.

**Worked Example:**
Consider the $2 \times 2$ matrix $A$:
$$A = \begin{pmatrix} 3 & 1 \\ 2 & 2 \end{pmatrix}$$
To find the eigenvalues, we solve $\det(A - \lambda I) = 0$:
$$\det\left(\begin{pmatrix} 3-\lambda & 1 \\ 2 & 2-\lambda \end{pmatrix}\right) = (3-\lambda)(2-\lambda) - (1)(2) = 0$$
$6 - 5\lambda + \lambda^2 - 2 = 0$, which simplifies to $\lambda^2 - 5\lambda + 4 = 0$. Factoring yields $(\lambda - 4)(\lambda - 1) = 0$. The eigenvalues are $\lambda_1 = 4$ and $\lambda_2 = 1$.
For $\lambda_1 = 4$, we solve $(A - 4I)v = 0$:
$$\begin{pmatrix} -1 & 1 \\ 2 & -2 \end{pmatrix} \begin{pmatrix} v_1 \\ v_2 \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \end{pmatrix}$$
Both rows yield the equation $-v_1 + v_2 = 0$, so $v_1 = v_2$. The eigenvector corresponding to $\lambda_1=4$ is thus $v_1 = k\begin{pmatrix} 1 \\ 1 \end{pmatrix}$.
