### angle (19.8s)

## Angle

每当你打开一扇门，门与门框之间就形成了一个张开的程度——这就是角。从时钟的指针到篮球的投篮弧线，角无处不在，它是几何学最基本的度量之一。

**定义。** 两条共享同一端点的射线所构成的图形称为**角**（angle）。共享的端点称为**顶点**（vertex），两条射线称为角的**边**（sides）。若射线 $\overrightarrow{OA}$ 与 $\overrightarrow{OB}$ 共享顶点 $O$，则所成的角记作 $\angle AOB$。角的大小度量一条边旋转到另一条边所经过的量，常用**度**（degree）或**弧度**（radian）表示：

$$360° = 2\pi \text{ rad}, \quad 1° = \frac{\pi}{180} \text{ rad}$$

按大小分类：锐角满足 $0° < \theta < 90°$，直角 $\theta = 90°$，钝角 $90° < \theta < 180°$，平角 $\theta = 180°$。

**例题。** 时钟在 3:00 时，时针指向 3，分针指向 12。钟面共 12 格，每格对应 $\frac{360°}{12} = 30°$。时针与分针相差 3 格，因此所成角为：

$$\theta = 3 \times 30° = 90°$$

这是一个直角。

**为什么弧度制成立？** 在半径为 $r$ 的圆中，弧度定义为弧长与半径之比：$\theta = \frac{s}{r}$。当弧长恰好等于半径时，$\theta = 1$ rad。绕整圆一周的弧长为 $2\pi r$，所以完整旋转对应 $\frac{2\pi r}{r} = 2\pi$ rad。这不是人为约定，而是圆的几何性质的自然结果——弧度制直接将角与弧长联系起来，使得微积分中的公式（如 $\frac{d}{d\theta}\sin\theta = \cos\theta$）不需要额外的转换系数。

**练习。** 将 $\theta = 150°$ 转换为弧度，并判断它属于哪一类角。

### distance (19.1s)

## Distance

在日常生活中，我们不断地衡量"远近"——从家到学校有多远？两座城市之间的距离是多少？这种对"间隔"的直觉，正是数学中**距离**概念的起点。

**定义。** 设 $X$ 是一个非空集合。函数 $d: X \times X \to \mathbb{R}$ 若满足以下三条公理，则称 $d$ 为 $X$ 上的一个**距离函数**（或**度量**）：

1. **非负性与同一性：** $d(x, y) \geq 0$，且 $d(x, y) = 0 \iff x = y$
2. **对称性：** $d(x, y) = d(y, x)$
3. **三角不等式：** $d(x, z) \leq d(x, y) + d(y, z)$

这三条公理刻画了我们对"距离"最本质的期待：距离不能为负，来回路程一样长，走直线不会比绕路更远。

**例题。** 在 $\mathbb{R}^2$ 中，取欧几里得距离 $d(\mathbf{p}, \mathbf{q}) = \sqrt{(p_1 - q_1)^2 + (p_2 - q_2)^2}$。计算 $\mathbf{p} = (1, 2)$ 与 $\mathbf{q} = (4, 6)$ 之间的距离：

$$d(\mathbf{p}, \mathbf{q}) = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

**为什么三角不等式成立？** 对于欧几里得距离，三角不等式的本质是**柯西-施瓦茨不等式**。直观地说，连接两点的线段是最短路径——经过任何中间点 $y$ 的折线 $x \to y \to z$，其总长度不可能比直线 $x \to z$ 更短。这不是巧合，而是欧几里得空间内积结构的必然结果。

值得注意的是，距离的定义并不依赖于坐标系。曼哈顿距离 $d_1(\mathbf{p}, \mathbf{q}) = |p_1 - q_1| + |p_2 - q_2|$ 同样满足这三条公理，只是度量了不同的"远近"。同一个集合上可以定义多种距离，每种距离赋予空间不同的几何结构。

**练习。** 验证曼哈顿距离 $d_1$ 对点 $\mathbf{a} = (0,0)$、$\mathbf{b} = (3,4)$、$\mathbf{c} = (1,1)$ 满足三角不等式，即 $d_1(\mathbf{a}, \mathbf{c}) + d_1(\mathbf{c}, \mathbf{b}) \geq d_1(\mathbf{a}, \mathbf{b})$。比较此结果与欧几里得距离下的情况。

### line (18.3s)

## Line

想象一条笔直的公路，向两端无限延伸，没有起点，也没有终点。这就是几何中**直线**的直觉来源——它是空间中最基本的对象之一。

**定义。** 在欧几里得几何中，直线是由两个不同的点唯一确定的一维对象，沿两个方向无限延伸。给定平面上两点 $A(x_1, y_1)$ 和 $B(x_2, y_2)$（其中 $A \neq B$），通过它们的直线可以用参数方程表示：

$$\mathbf{r}(t) = (1-t)\,\mathbf{A} + t\,\mathbf{B}, \quad t \in \mathbb{R}$$

当 $x_1 \neq x_2$ 时，直线也可以写成斜截式 $y = mx + b$，其中斜率 $m = \dfrac{y_2 - y_1}{x_2 - x_1}$。

**例题。** 求过点 $P(1, 2)$ 和 $Q(4, 8)$ 的直线方程。

先算斜率：

$$m = \frac{8 - 2}{4 - 1} = \frac{6}{3} = 2$$

代入点斜式 $y - y_1 = m(x - x_1)$：

$$y - 2 = 2(x - 1) \implies y = 2x$$

验证：当 $x = 4$ 时，$y = 8$，与 $Q$ 吻合。✓

**为什么两点确定一条直线？** 这是欧几里得第一公设的内容。直觉上，如果存在两条不同的直线都经过 $A$ 和 $B$，则它们之间的区域会产生"弯曲"，与直线的定义矛盾。在公理化体系中，这一性质被直接假定为基础真理——所有后续定理都建立在它之上。

直线的概念延伸至高维空间：在 $\mathbb{R}^n$ 中，过点 $\mathbf{p}$、方向为 $\mathbf{d} \neq \mathbf{0}$ 的直线为 $\{\mathbf{p} + t\,\mathbf{d} \mid t \in \mathbb{R}\}$。无论维度如何变化，"由一点和一个方向唯一确定"这一本质不变。

**练习。** 直线 $\ell$ 过点 $(−1, 5)$ 且斜率为 $-3$。求 $\ell$ 与 $x$ 轴的交点坐标。
