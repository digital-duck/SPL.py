## Function

A function is a fundamental concept in advanced mathematics that formalizes the relationship between two sets, ensuring that every element of one set corresponds to exactly one element of another. Formally, let $A$ and $B$ be non-empty sets. A function $f$ from $A$ to $B$, denoted $f: A \to B$, is a rule that assigns to each element $x \in A$ (the domain) precisely one element $y \in B$ (the codomain). The set of all actual outputs $\{f(x) | x \in A\}$ is called the *range* of $f$. It is crucial to note that the range is a subset of the codomain; thus, $\text{Range}(f) \subseteq B$.

**Worked Example:** Consider the function $f: \mathbb{R} \to [0, \infty)$ defined by $f(x) = x^2 + 1$. Here, the domain is the set of all real numbers ($\mathbb{R}$), and the codomain is $[0, \infty)$. For any input $x$, the output $y$ must be in $[0, \infty)$, which holds true since $x^2 \ge 0$, meaning $x^2 + 1 \ge 1$. Therefore, while the range of this function is actually $[1, \infty)$, it still adheres to the stated codomain $[0, \infty)$. This example demonstrates that the codomain merely sets the potential bounds for the output, while the domain dictates all valid inputs.

### limit (0.2s)

## Limit

The concept of a limit formalizes the notion of approach, providing a rigorous tool to analyze the behavior of functions near specific points, even if those points are excluded from the function's domain. Formally, we say that the limit of a function $f(x)$ as $x$ approaches a value $a$, written $\lim_{x \to a} f(x) = L$, if for every positive number $\epsilon > 0$, there exists a corresponding $\delta > 0$ such that if $0 < |x - a| < \delta$, then $|f(x) - L| < \epsilon$. This definition, known as the $(\epsilon, \delta)$ definition, is fundamental because it replaces intuitive notions of "getting close" with quantifiable inequalities. It establishes that the value $L$ must be arbitrarily close to $f(x)$ whenever $x$ is sufficiently close to $a$, but not equal to $a$.

Consider the function $f(x) = \frac{x^2 - 4}{x - 2}$. This function is undefined at $x=2$. To determine its limit as $x$ approaches 2, we analyze the algebraic structure. For $x \neq 2$, we can factor the numerator:
$$f(x) = \frac{(x - 2)(x + 2)}{x - 2} = x + 2.$$
Since the simplified form $g(x) = x+2$ is continuous everywhere, its limit at $x=2$ must be substitutionally valid. Thus, $\lim_{x \to 2} f(x) = 2 + 2 = 4$. The existence of this limit demonstrates that the function approaches a finite value even though it has a removable discontinuity at $x=2$.

A key theorem involving limits is the Limit Sum Theorem: if $\lim_{x \to a} f(x)$ and $\lim_{x \to a} g(x)$ both exist, then $\lim_{x \to a} [f(x) + g(x)] = \lim_{x \to a} f(x) + \lim_{x \to a} g(x)$. This theorem allows us to decompose the computation of complex limits into simpler parts. By repeatedly applying linearity properties (e.g., $\lim [c \cdot f(x)] = c \cdot \lim f(x)$), we can evaluate many types of indeterminate forms.

Formally, given a real-valued function $f(x)$, its derivative, denoted by $f'(x)$ or $\frac{d}{dx} f(x)$, measures how rapidly the corresponding output value changes with respect to its input variable at any specific point $x$. ### gradient (0.2s)

## Gradient

The gradient, denoted by the symbol $\nabla$, is a fundamental operator in multivariable calculus that maps a differentiable scalar field $f: \mathbb{R}^n \to \mathbb{R}$ into a vector field. Geometrically and analytically, it quantifies the rate and direction of maximum change of the function $f$. For a function $f(x_1, x_2, \ldots, x_n)$, the gradient $\nabla f$ is defined as the vector whose components are the partial derivatives of $f$:
$$ \nabla f = \left\langle \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n} \right\rangle $$
In two dimensions, for a function $f(x, y)$, the gradient is calculated as:
$$ \nabla f = \left\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right\rangle $$
The magnitude of this vector, $\|\nabla f\|$, represents the maximum rate at which $f$ changes at a given point. The direction of $\nabla f$ indicates the path along which the function increases most rapidly, known as the direction of steepest ascent.

## Gradient Descent

Gradient Descent (GD) is a foundational iterative optimization algorithm used to locate the local minima of a differentiable objective function $J(\mathbf{w})$. Given a loss landscape defined by $J(\mathbf{w})$, where $\mathbf{w} \in \mathbb{R}^d$ represents the model parameters, GD aims to minimize this function without requiring explicit inversion of the Hessian matrix. The core mechanism relies on the principle that moving in the direction opposite to the gradient (the steepest ascent) guarantees a local decrease in the function value. Mathematically, the update rule for the parameters $\mathbf{w}$ at iteration $t+1$ is given by:
$$ \mathbf{w}_{t+1} = \mathbf{w}_t - \eta \nabla_{\mathbf{w}} J(\mathbf{w}_t) $$
where $\eta > 0$ is the learning rate, controlling the step size, and $\nabla_{\mathbf{w}} J(\mathbf{w})$ is the gradient vector of the loss function with respect to $\mathbf{w}$.
