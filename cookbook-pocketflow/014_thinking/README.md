**Difficulty:** ★☆☆ Beginner
**SPL pattern:** Iterative CoT with inner retry loop — nested `WHILE` + YAML validation

## What it demonstrates

Solves complex reasoning problems through multi-step chain-of-thought, where
each thought step is validated as YAML before being accepted.  The inner
`WHILE` loop retries up to 3 times if the LLM produces malformed output —
a reusable robustness pattern for structured generation.

Key SPL constructs:
- Outer `WHILE @next_thought_needed = "true"` — reasoning loop
- Inner `WHILE @yaml_valid = "false"` — YAML validation retry
- `GENERATE ChainOfThoughtStep(…)` — single reasoning step
- `CALL validate_yaml_fields / extract_yaml_field` — structured output tools
- `CALL write_file(…, "a")` — append-mode trace logging
- `EXCEPTION WHEN BudgetExceeded` — graceful budget handling
