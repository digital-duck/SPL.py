## What it demonstrates

A web-research agent that iterates decide → search → accumulate until it has
enough context to answer a question.  This is the canonical **ReAct** pattern
in SPL: the LLM decides the next action via structured YAML output; the
workflow dispatches to a tool (`search_web`) or terminates.

Key SPL constructs:
- `WHILE … DO … END` — bounded iteration with a `done` flag
- `GENERATE decide_action(…)` — LLM produces YAML action decision
- `CALL parse_yaml / extract_field` — structured output parsing tools
- `EVALUATE … WHEN contains(…)` — branch on LLM output content
- `CALL search_web` — external tool invocation
- Nested `EVALUATE` for YAML parse-error retry
