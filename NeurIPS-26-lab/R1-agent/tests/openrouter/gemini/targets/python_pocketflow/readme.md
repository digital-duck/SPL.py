### Setup Instructions
1. Install Python 3.8+.
2. Install `requests`: `pip install requests`.
3. Set your environment variable: `export OPENROUTER_API_KEY='your_key_here'`.
4. Ensure you have network access to `openrouter.ai`.

### Run Command
```bash
python S3-agent-openrouter-gemini.py
```

### Expected Output Pattern
```text
Status: complete
Iterations: 1
Answer: [A detailed response generated by Gemini based on the simulated search context]
```

### SPL to PocketFlow Mapping Table

| SPL Construct | PocketFlow / Python Equivalent |
| :--- | :--- |
| `CREATE FUNCTION` | Python function with f-string prompt and LLM API call |
| `WORKFLOW` | Class inheriting from `Workflow` (or standard class) |
| `INPUT` | `__init__` parameters and `self.state` initialization |
| `WHILE` | Standard Python `while` loop |
| `GENERATE` | Function call to LLM wrapper |
| `EVALUATE` | Python `if/elif/else` logic |
| `CALL` | Standard Python function call |
| `RETURN ... WITH` | Return dictionary containing primary value and metadata |
| `@variable` | Keys within `self.state` dictionary |
| `:=` (Assignment) | Dictionary value update or variable assignment |
