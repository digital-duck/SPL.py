### Setup Instructions
1. Install Python 3.9+.
2. Install dependencies: `pip install requests`.
3. Set your OpenRouter API Key: `export OPENROUTER_API_KEY='your_key_here'`.
4. Save the code as `s3_agent_openrouter_gemini.py`.

### Run Command
```bash
python s3_agent_openrouter_gemini.py
```

### Expected Output Pattern
The script will iterate up to 5 times, calling Gemini to decide if it needs more info. It will then print a synthesized response:
```text
Final Response: [Comprehensive answer generated by Gemini...]
Status: complete
```

### SPL to PocketFlow Mapping Table

| SPL Construct | PocketFlow / Python Implementation |
|:---|:---|
| `CREATE FUNCTION` | Python function returning a formatted string to `call_llm` |
| `WORKFLOW` | Class inheriting from `Workflow` (or standard Python class) |
| `INPUT` | Arguments in the `run` method |
| `@variable := value` | Standard Python variable assignment |
| `WHILE ... DO` | `while` loop with boolean logic |
| `GENERATE func() INTO @var` | `var = func_wrapper(args)` (calling LLM) |
| `EVALUATE ... WHEN` | `if / elif / else` conditional blocks |
| `CALL tool()` | Direct call to a Python helper function |
| `RETURN ... WITH` | Return a dictionary containing the output and metadata |
